<div><p>
 【大纪元2021年01月02日讯】（大纪元记者林丹、李文报导）学年期中，纽约一公校以疫情中确保班级人数和安全为由，突然宣布
 <ok href="https://www.epochtimes.com/gb/tag/%E5%A4%A9%E6%89%8D%E7%8F%AD.html">
  天才班
 </ok>
 与普通班级重组。这让家长忧心该校天才班的项目将被迫中断，以及对对孩子的心理造成伤害。
</p>
<p class="p1">
 纽约的Leal女士告诉大纪元记者，圣诞节假日前夕，
 <ok href="https://www.epochtimes.com/gb/tag/%E7%BA%BD%E7%BA%A6%E5%85%AC%E6%A0%A1.html">
  纽约公校
 </ok>
 220
 <ok href="https://www.epochtimes.com/gb/tag/%E5%A4%A9%E6%89%8D%E7%8F%AD.html">
  天才班
 </ok>
 一年级的学生家长收到一个简短通知，称为了确保确班级安全和班级规模，有必要重新配置班级，这意味着对某些学生而言，班级/老师会有改变。
</p>
<p class="p1">
 家长发现，根据班级的重新配置，该天才班的12名学生全部被转移到同年级一个普通班。
</p>
<p class="p1">
 同时，原天才班的老师宣布她因为健康原因成为一名
 <span class="s1">
  远
 </span>
 程教师，只能跟天才班的孩子们说再见。
</p>
<p class="p1">
 “我一直感觉头晕，明天我要去看医生了。”该老师说。因为在此之前，天才班的老师和学生们刚刚被居家隔离了14天，在校长耐心劝导家长接受校内
 <ok href="https://www.epochtimes.com/gb/tag/%E7%97%85%E6%AF%92%E6%A3%80%E6%B5%8B.html">
  病毒检测
 </ok>
 后，学校出现一名阳性确诊案例，这些孩子们被认定为密切接触者，只能
 <span class="s1">
  远
 </span>
 程上课。接到通知的这一天正是隔离期的最后一天。
</p>
<p class="p1">
 本来准备过年的多位家长们马上给校长写信，表达自己的担忧。
</p>
<p class="p1">
 家长认为，在学期中重新安排班级，对年幼的孩子来说是很困难的，应该考
 <span class="s1">
  虑
 </span>
 到
 <span class="s2">
  “创伤知情（
 </span>
 <span class="s3">
  trauma
 </span>
 <span class="s4">
  –
 </span>
 <span class="s3">
  informed
 </span>
 <span class="s2">
  ），而这突然的决定显示没有考
 </span>
 <span class="s5">
  虑
 </span>
 <span class="s6">
  儿童的情绪健康。
 </span>
</p>
<p class="p5">
 家长们对一些不可控因素（就读人数、老师的健康状况）表示理解，但希望校方与学生家长沟通，让孩子们在情感和心理上做好准备以应对这种变化。
</p>
<p class="p5">
 目前到校学生被分为a和b两组，轮流上课。家长希望孩子每周到校上四天课（周一被规定上网课），而不是与其它班级合并。
</p>
<p class="p5">
 此外，家长没有收到天才班G&amp;T program是否仍存在的相关信息。家长认为，如果G＆T班级与主流班级合并，将对这些孩子产生长期影响。
</p>
<p class="p5">
 家长寻求替代方案，能给孩子们在自己的课堂环境中继续提供G＆T课程。作为父母，家长们表示尽可能的与学校合作，以最大程度地减少对孩子的伤害，让孩子度过一学年。
</p>
<p class="p5">
 校长Pizarra很快回信了，她写道，“作为父母和教育者，我了解您的担忧。不幸的是，由于当前的情况以及疫情导致的班级规模问题，这是我们唯一的选择，以确保我们学校社区中的所有孩子都能在安全和严格的环境中继续学习。”
</p>
<p class="p5">
 校长称，管理层在11月份就与家长讨论了班级重组的问题。
</p>
<p class="p5">
 她强调，“对于天才班的孩子，所有学生仍保持优才计划的部分。班级结构的变化不会影响或改变孩子的资优状况。所有的孩子都将保留在他们当前的组中，我们正在向组中添加其他孩子。因此，孩子们彼此之间或与他们当前的同学没有分开。”
</p>
<p class="p5">
 此外，校长还提供了
 <span class="s1">
  远
 </span>
 程学习的选择，有一名G＆T老师将从事远程的一年级天才班教学。
</p>
<p class="p5">
 Leal介绍，这封信让家长的心稍事安心。转眼圣诞节过去了，又有家长写信给校长，质疑如果孩子不在G＆T班级，他们将如何获得G＆T课程？
</p>
<p class="p5">
 家长认为校方与家长缺乏沟通，孩子们需要过渡期。家长希望在开学之前校长与家长举行一次zoom会议，以交流沟通一些重要问题。但校长没有做出回应。
</p>
<p class="p5">
 <span class="s1">
  家长们
 </span>
 对纽约天才班的走向表示担忧。“家长不清楚天才班是否会继续，为此协议没有得到遵守而感到沮丧。”Leal认为，这是该校多名老师因为各种原因选择远程教学，导致了班级重组的直接后果。
</p>
<p class="p5">
 “家长表示理解当前的疫情，但是不应该损害孩子们的教育需求，学校需要在孩子们自己的课堂中提供G＆T课程。”她说。
</p>
<p class="p5">
 <span class="s1">
  公开资料显示，美国
 </span>
 <span class="s2">
  “
 </span>
 <span class="s1">
  天才班
 </span>
 <span class="s2">
  ”
 </span>
 <span class="s1">
  的历史颇为悠久，可以追溯到
 </span>
 <span class="s2">
  19
 </span>
 <span class="s1">
  世纪末。
 </span>
 据媒体报导，2019年8月，纽约
 <span class="s7">
  公校多元
 </span>
 <span class="s7">
  化小组做出
 </span>
 <span class="s8">
  建议取消天才班的报告，认为该项目中
 </span>
 <span class="s7">
  中亚裔和白人学生比重过大，这对其他族裔来说有失公平，造成“族裔隔离”。
 </span>
</p>
<p class="p7">
 <span class="s9">
  此报告随即引发反对声浪。纽约市议员顾雅明也发声明表示谴责，指出这些天才班是全市很多家长梦寐以求的教育资源，不仅不应被砍掉，反而应该扩展，“增加多元化和扩展高中教育不应该以成功作为替罪羊。”
 </span>
</p>
<p class="p9">
 <span class="s7">
  然而2020年的天才班考试（包括
 </span>
 <span class="s10">
  特殊高中SHSAT考试
 </span>
 <span class="s7">
  ）却因疫情原因被推
 </span>
 <span class="s11">
  迟
 </span>
 <span class="s7">
  了，家长多次对此表示抗议。
 </span>
</p>
<p class="p11">
 Leal说，“由于天才班的设置不限学区，有家长住在很
 <span class="s1">
  远
 </span>
 的地方，每天开车送孩子上学。如果没有天才班了，她会选择附近的学校就读。她原以为选择这所天才班学校是最好的选择，所以希望能给孩子们一个好的解决方案。”
</p>
<p class="p1">
 <span class="s1">
  记者致信220学校校长和纽约市教育局寻问班级重组等问题，220校长没有回应。教育局回复称错过了记者的信件，故暂无回应。
 </span>
</p>
