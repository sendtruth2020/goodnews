 <p>
  【看中国2021年5月22日讯】(看中国记者真瑜采访报道）5月18日午间，深圳“地标”79层的
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/赛格大厦" target="_blank">
    赛格大厦
   </span>
   突发晃动
  </strong>
  导致大厦内人员四散而逃，当时人们误以为是发生了
  <strong>
   地震
  </strong>
  。大陆官方机构在18日晚立刻发布通告称大楼摇晃系偶发原因。然而，打脸的是大厦于19日、20日两日再次发生了摇晃。
  <br>
   <br>
    目前赛格大厦进入封闭状态，暂停所有业主、商户、租户进出赛格大厦写字楼和电子市场，相关部门人员开始运输施工材料疑似紧急修复大楼，但官方并未就此做出解释。21日，经大陆专家测算，大厦各项监测数据正常，未超出相应标准要求。因为当日大厦已经封闭，所以大厦是否发生第四次摇晃目前无法证实。
    <br>
     <br>
      大陆官方公布了赛格大厦房屋沉降、倾斜率和加速度监测结果，三项数据都显示正常，专家特地指出大厦无振感，以此安抚民心。5月18日下午7点，深圳住房和建设局发布测结果表明大厦摇晃是偶发原因，官方在事发数小时后就宣布楼体安全不知道如何在这么短的时间内完成检测。而后大陆中冶建筑研究总院（深圳有限公司）及哈尔滨工业大学（深圳）两家机构联合作保说大楼安全无虞。
      <br>
       <br>
        有大陆专家甚至将大厦的摇晃归咎于贯穿于大厦底部的所谓地铁，更有甚者说大厦摇晃是因为大厦内使用了大量的硬盘进行虚拟币挖矿，由于硬盘产生的共振导致大厦摇晃。专家提供的理由千奇百怪，就是不提楼体的质量问题导致摇晃。
        <br>
         <br/>
         其实早在20年前就有人断言大楼会发生摇晃，20年前一篇题目为《深圳赛格广场建设项目评析》的硕士论文中就指出，大厦是“边设计边施工”，1999年，正直中共国庆节期间，赛格大厦楼顶天线剧烈摇摆。此时距离大楼真正竣工不到7天，经查，原因是天线设计计算错误造成共振。
         <br/>
         <br/>
         之所以论文能指出大厦的种种问题，是因为论文作者金典琦参与了赛格大厦从规划到建造的全过程。当时她以在职研究生身份在华中科技大学攻读硕士学位。她也非建筑学科班出身，可以说是个参与了大厦建设项目的局外人。结果恰恰是局外人的调研秒杀了科班专家的权威性，也许是脸打得太狠，金典琦的硕士导师说该论文只能表示作者有想法，但不能代表科学。
         <br/>
         <br/>
         赛格大厦是深圳的地标建筑，当时其建设速度惊人，被称为是深圳速度、中国速度，只是速度背后的质量问题被掩盖了。据赛格大厦高层的办公人员称，大厦在中午的时候摇晃剧烈，甚至一度倾斜，人在摇晃的大楼中感觉到头晕恶心，原本人们以为是心理作用，待到有明显的倾斜后大家纷纷开始逃命，其实大楼也并非今年才发生晃动，只是之前的晃动较为轻微。
         <br/>
         <br/>
         记者为此也特地访问了建筑行业从业超过10年的设计师，该设计师表示，大陆的商业建筑存在很严重的边建设边设计问题，比如说施工方提交给官方审批的是一个设计图，等到建设时就要改动，以此达到利益的最大化，施工方因为私自更改设计图纸造成与设计师间的“相互扯皮”是家常便饭。
         <br/>
         <br/>
         另外，大陆的监理制度形同虚设，基本都是走走过场，这就造成了很多建筑都存有瑕疵，也就是人们说的“豆腐渣”工程。尤其是沿海地区，违规使用海沙的现象比较严重，海沙与混凝土混合后的水泥质量不到正常的50%，如果钢筋再有问题，楼体倾覆、坍塌只是时间问题。
         <br/>
         <br/>
         还有，大陆建商常用包商施工，就是说项目承接方与施工方很多时候不是一伙人，层层转包下牺牲的都是建筑的质量，大建筑商找小的施工方，小的施工方再把活包给更小的公司，最后导致出了质量问题维权都成了拉锯战。
         <br/>
         <br/>
         从政府的角度看，政府官员为了政绩非常希望建筑方能迅速完成项目增加地方的GDP，为什么当初万达在地方大受欢迎，就是因为万达保证在地方一把手任期之内完成项目，大建商如同万达也许在保证速度的同时还能兼顾质量，小建商则完全没有这个能力。
         <br/>
         <br/>
         另外，大厦如果发生质量问题，后续的检测工作十分复杂，根本不可能在短短几天就完成，需要用专业的工具勘测楼梯的主体部位，需要使用探测器探测承重墙内部的钢筋是否有断裂或是发生形变，而且是每个楼层都要注意排除，不知道官方如何在2天内就完成了79层大厦的质量复查。
         <br/>
         <br/>
         总之，赛格大厦摇晃官方怪风、怪地铁、怪气温、怪硬盘、怪人心作怪、怪人民刁难，就是不怪他们的楼质量有问题，如果质量没问题，为何又要运送建筑材料进入大楼，总之好奇怪。
