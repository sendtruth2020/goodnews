  【看中国2021年5月12日讯】近日，大陆发生一件极其令人捉摸不透的“
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/坠楼事件" target="_blank">
    坠楼事件
   </span>
  </strong>
  ”，
  <strong>
   成都成华区49中学
  </strong>
  高二学生
  <strong>
   林某某
  </strong>
  于上周日
  <strong>
   坠楼死亡
  </strong>
  ，据了解死亡者林某某平时学习成绩优异，官方调查组仅用2天时间就完成了“坠亡事件”的调查认定，官方认为高空坠亡属个人行为，排除刑事案件，未发现存在体罚、辱骂等师德失范问题，未发现林某某受到校园欺凌情况，林某某死亡属于其心理发生问题是单纯的个别事件。
  <br>
   <br>
    与官方不同的是，民间认为林某某的死亡背后另有原因，5月9日本是母亲节，林某某的母亲却在母亲节当日失去了爱子，更令人悲伤的是周一还抱着林某某遗像在学校门口鸣冤的林妈妈突然改变口风，称对林某某的死亡没有异议。
    <br>
     <br>
      林家短短数日接连遭受两次打击，先是孩子莫名其妙死亡，紧接着是被封口，媒体被禁止采访一律使用官方通报，真相也许再也无法查证，官方处理之冷漠无情，更是为该事件写上一个大大的“惨”字。
      <br>
       <br>
        比“冤”更甚的就是“惨”，5月9日下午5点40分林某某到达学校，晚上9点学校突然来电说林某某已经死亡，林某某真实的死亡时间是下午6点49分，也就是说林某某进校1小时后就发生了意外，学校没有第一时间通知家属而是拖到了晚上9点多。
        <br>
         <br/>
         家属随后分别赶往殡仪馆和学校，此时家属内心一连串问号但就是得不到答案，家属被关在门外不让去事发地勘验，家属想看监控却被告知监控失灵无法查看，等看到监控的时候偏偏少了最关键的一段；林某某的同学都被遣散回家，并且被禁止谈论此事。林某某的死亡更是扑朔迷离，监控少一段，直接殡仪馆，笔者非常疑惑。在关键证据缺失的情况下官方是怎么判断学校和老师是无责任的。

 <p>
  周日
  <span href="https://www.secretchina.com/news/gb/tag/林某某" target="_blank">
   林某某
  </span>
  去学校前还对林母说母亲节快乐，哪成想再见竟是永别。学校的操作堪比教科书，不知道是不是从08年地震中学到了隐瞒不报的精髓，此番49中的学校领导者将欺上瞒下发挥的淋漓尽致。
  <br/>
  <br/>
  据成都49中的学生复述，事发初始学校突然派人到各班通知学校在打农药，任何学生只能待在教室内不得外出，林某某同班的同学也被下达封口令不许谈论有关事件，凡对外透露此事者一律开除且不会被他校录取且无法参加高考，这是赤裸裸的威胁。
  <br/>
  <br/>
  林某某6点49分坠楼后，校方不知出于何种原因既没立刻报警也没立即求医，可怜的林某某就静静在楼下任由鲜血流出的躺了一个多小时，直到8点多尸体才被救护车拉走，最后林某某的尸体没有放入医院待法医检查而是直接送到了殡仪馆，也就是说林某某自离家上学后，其家人就再也没有见到他，现在林某某的尸体依旧在殡仪馆，另有说法称林某某的尸体在家属未签字的情况下被官方强行火化。
  <br/>
  <br/>
  目前，网络中凡对该事件有疑问的，包括爆料在内的信息全部被删除，林母的微博也被清空，林母的手机号也处于无人接听的状态，官方通报说家属没有异议，但是家属到学校门口维权时被强行拖走的视频明显与官方说法相左。
  <br/>
  <br/>
  媒体被禁声，民间不让议论，官方一片家属情绪稳定，家属至今联系不上，学校一翻不打自招式的操作都在说明一个问题，事出反常必有妖，对于官方的通报，只能一句概括，“我信你个鬼”。
  <br/>
  <br/>
  民间的说法似乎更有说服力，林某某的死亡其实是出于该校语文老师同时兼任教研组主任万利霞所为，因为该校将有一个出国留学名额被成绩优秀的林某某获得，因为嫉妒同时想让林某某让出名额给自己的孩子，于是万老师恶从心中起，最终导致林某某死亡，也有说法称万利霞本意只是吓唬林某某，结果不小心将其推倒致其死亡，目前万利霞老师并没有被警方控制。该事件已经引起很大民愤，很多媒体不顾禁令纷纷跟进报道，许多学生家属纷纷站出来支持林妈妈查清真相。
  <br/>
  <br/>
  如果这么看就是简单的一起刑事案件，但是学校面面俱到，为了维护学校声誉，为了保住各位领导的官位，为了不给校内的高三学生制造心理压力，为了他们心中的这些那些，第一时间不是救人命而是联合官方掩盖事实，唯独忘记了学校是教书育人之地，学生才是学校的重中之重，学校的本末倒置除了秀了学校处理应急事件能力，就是再度拉低了人们的道德底线。
