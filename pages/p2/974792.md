 <p>
  【看中国2021年6月12日讯】近日，美国情报界人士透露，“史上最高级别中国（中共）
  <strong>
   叛逃
  </strong>
  者”与美国国防情报局（DIA）合作3个月，此人向美方和盘托出关于中国的“特殊武器计划”，包括“生化武器计划”。事件引发美国一系列连锁反应，同时对
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/中南海" target="_blank">
    中南海
   </span>
  </strong>
  造成冲击。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  据美联社6月4日报导，美国《福克斯新闻》主播卡尔森表示，一位美国情报界内部消息人士匿名向保守派网站“红州”透露，一名“有史以来最高级别的出逃者”，已经与美国国防情报局DIA合作了3个月。这位出逃者向美方透露了中共“生物武器计划”等消息。
  <br>
   <br>
    “红州”独家引述消息人士的说法，指这位出逃者对中共的特殊武器，包括生物武器计划“有直接了解”。国防情报局领导层将出逃者留在了他们的秘密服务网路中，以防止中央情报局和国务院接触到这个人。原因是国防情报局领导层认为，FBI、CIA和其它的几个机构内部有中共间谍。
    <br>
     <br>
      消息人士还告诉“红州”，出逃者已经在国防情报局工作了3个月，向美国官员提供了广泛的、技术上的详细汇报。而国防情报局经过评估，认定出逃者提供的信息是可靠的。
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  曾获美国艾美奖的记者豪斯利核实了史上“最高级别中国（共）叛逃者”的消息。他在推特上指出，美方情报人员称，中共试图制造“
  <span href="https://www.secretchina.com/news/gb/tag/病毒" target="_blank">
   病毒
  </span>
  来自蝙蝠”的表象，以掩盖它来自实验室的事实。
 </p>
 <p>
  众所周知，历史上叛离中共最高级别官员，当属“六四”时期的新华社香港分社社长许家屯。正部级官员。现在的机构叫做中联办。许家屯没来香港之前是江苏省委第一书记。
 </p>
 <p>
  1989“六四事件”期间，许家屯曾去看望在香港新华社门前静坐声援北京学运的香港学生，他向中共中央提出与和平请愿的学生对话，顺势推动政治改革的建议。“六四事件”发生后，许家屯看到赵紫阳被整肃，同样的命运正在等待着他，于是三十六走为上，决定登上飞机跨洋过海出走美国，被中共定性为“
  <span href="https://www.secretchina.com/news/gb/tag/叛逃" target="_blank">
   叛逃
  </span>
  ”。他在美国度过了26个春秋，于2016年6月在洛杉矶奇诺岗家中逝世，终年100岁。
 </p>
 <p>
  有分析指，如果这次叛逃的真是有史来最高级别，那便是副国级。且一定是军方负责生物战方面的高级将领，战区司令以上、军委委员这个级别，相当于副国级以上的军官。
  <br>
   <br>
    <strong>
     美国六大变化
    </strong>
    <br>
     <br/>
     前中共政治局常委、中纪委书记尉健行的撰稿人王友群也指出，从美国主流媒体的报导看，这个“史上最高级别”的中共叛逃者带来的情报，可能与“中共病毒”（COVID-19）起源有关。而近期，美国政府及国际社会围绕“中共病毒”源头问题，也出现了一系列重大变化。
     <br/>
     <br/>
     第一、美国总统
     <span href="https://www.secretchina.com/news/gb/tag/拜登" target="_blank">
      拜登
     </span>
     的态度变了。
     <br/>
     <br/>
     5月26日，拜登发表声明，下令美国情报机关90天内向他提交病毒起源的报告。在此之前，中共病毒溯源没有列入拜登重要的议事日程上。而且关于病毒是从实验室泄露的说法曾受到主流媒体排斥。
     <br/>
     <br/>
     第二、美国国务卿布林肯的态度变了。
     <br/>
     <br/>
     布林肯在6月6日播出的美国新闻节目“Axios on HBO”中说，拜登政府决心“彻查”中共病毒起源，并将追究中共的责任。在此之前，布林肯3月28日接受CNN采访时曾表示，拜登政府不太会在对待疫情爆发的事情上惩罚中共。
     <br/>
     <br/>
     第三、美国总统首席医疗顾问福西的态度变了。
     <br/>
     <br/>
     美国顶级传染病和公共卫生专家福西5月24日接受福克斯新闻网采访时说，“我并不确信病毒来自大自然，我认为我们应该继续调查中国到底发生了什么。”
     <br/>
     <br/>
     CNN称，福西亲自推翻了他一年前的观点。一年前，他认为病毒是从动物传给人类的，而不是实验室泄漏的。
     <br/>
     <br/>
     最近，《华盛顿邮报》等媒体通过美国《信息自由法》，获得了福西3234页私人电子邮件（从2020年1月到6月）。
     <br/>
     <br/>
     邮件内容显示，福西与中共疾控中心高福关系密切，他有帮助中共掩盖病毒从实验室泄漏的痕迹。福西因此陷入信任危机，要求他辞职的呼声不断。
    </br>
   </br>
  </br>

 <p>
  6月7日，美国前总统
  <span href="https://www.secretchina.com/news/gb/tag/川普" target="_blank">
   川普
  </span>
  (特朗普)（特朗普）在福克斯节目中表示，福西已经落入了中共的口袋。
  <br/>
  <br/>
  川普说，“看看他的电子邮件，他们真的很可怕。看起来福西已经落在了中共的口袋里，他的处理方式都是在迎合中共。”
  <br/>
  <br/>
  第四、美国国会议员对中共的追责声更强了。
  <br/>
  <br/>
  5月26日，美国参议院通过议案，要求国家情报总监在90天内，解密有关武汉病毒研究所与中共病毒起源潜在关联的所有情报。
  <br/>
  <br/>
  5月28日，众议院209名共和党议员致信议长佩洛西，要求国会彻查中共病毒起源，并让中共为隐瞒疫情担责。如果证实病毒是实验室泄漏的，“中共就要为将近60万美国人和全世界数百万人的死亡负责”。
  <br/>
  <br/>
  第五、国际科学界对病毒溯源的呼声更高了。
  <br/>
  <br/>
  5月13日，来自美国、加拿大、英国、瑞士等国的18位世界顶级科学家在《科学》杂志发表公开信，呼吁对病毒的所有可能起源进行全新的独立调查。
  <br/>
  <br/>
  第六、美国主流媒体的态度也变了。
  <br/>
  <br/>
  5月23日《华尔街日报》援引一份此前未披露的美国情报机构的报告说，武汉病毒研究所有三名研究人员2019年11月染病就医。这个举动比中共官方披露新冠病毒传播的时间早一个多月。
  <br/>
  <br/>
  该报导称，这份报告内容包含受影响的研究人员的数量丶他们患病的时间、在医院就诊的细节等。报告细节可能会增加外界呼吁对病毒是否从实验室泄漏进行更广泛的调查。
  <br/>
  <br/>
  这则报导引发热议，美国多家主流媒体也通通改变口径，认为实验室病毒泄漏理论有一定可信性，应得到进一步调查。
  <br/>
  <br/>
  5月26日，脸书发布声明表示，将不再从平台上删除“新冠病毒为人造”的内容。
  <br/>
  <br/>
  为什么会发生上述一连串重大变化？王友群认为，很可能与“史上最高级别的”中共出逃者有关。最大的一种可能性是：这名中共出逃者，向美国提供了实锤的重磅情报。这一点从中共对病毒溯源调查的激烈反应中，也可得到反证。目前，中共正面临国际社会追责、索赔的巨大压力.
