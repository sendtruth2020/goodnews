  【看中国2020年11月17日讯】中共中央组织部部长
  <strong>
   陈希

  是习近平的“铁哥们”之一，陈希不但因为是
  <strong>
   习近平

  当年的清华同窗，而且陈希掌管
  <strong>
   清华大学

  期间，习近平获得了博士学位。
  <br>
   <br>
    1975年，
    <span href="https://zh.wikipedia.org/zh-hans/%E4%B9%A0%E8%BF%91%E5%B9%B3">
     习近平
    </span>
    获得推荐，进入清华大学化工系基本有机合成专业学习。在大学期间，习近平结识了同为化工系工农兵学员的上下铺室友陈希，两人同龄同系，且同是体育迷，使得他们培养深厚的友谊。

  </br>
 </p>
 <p>
  外界有所不知的是，
  <span href="https://www.secretchina.com/news/gb/tag/习近平" target="_blank">
   习近平
  </span>
  和陈希之间“铁哥们”关系的一个最重要情节是陈希是习近平的入党介绍人之一。由于父亲习仲勋被打成反党集团的一员，习近平被归类为“黑帮子弟”，习早年曾经9次申请入党被拒，至第10次申请入党时，习近平得到在清华大学担任学生党支部书记的陈希作为两名入党介绍人之一，习才得以加入中共。
  <br>
   <br>
    1979年3月，陈希与习近平同时离开清华校门，被安排回到自己原籍福建省的一所大学任教。当年夏天，陈希即考取清华研究生，回京报道后的第一件事情就是拜访已经在军委办公厅穿上了四个兜儿的军装的习近平。当年中共部队的军装没有军衔标志，上衣四个兜儿的是“干部服”，士兵的上衣只有两个兜儿。
    <br>
     <br>
      后来，习近平在清华大学读博士，据说就是陈希在幕后牵线搭桥。在习近平主政浙江时，还与已当上清华大学党委书记的陈希多有合作，通过两人努力浙江省和清华大学创建了浙江清华长三角研究院。
      <br>
       <br/>
       中共官方公布的习近平和陈希二人的简历显示，习近平是1998－2002年在清华大学人文社会学院马克思主义理论与思想政治教育专业在职研究生班学习，获法学博士学位，而陈希则是在美国斯坦福大学做了两年访问学者后于1993年开始担任清华大学党委副书记。
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  习近平在职读博开始的那一年，
  <span href="https://www.secretchina.com/news/gb/tag/陈希" target="_blank">
   陈希
  </span>
  被晋升半格，成为清华大学党委常务副书记，习近平完成在职读博的2002年，陈希被晋升清华大学党委书记，官至副部长级。
  <br/>
  <br/>
  2007年习近平在中共十七大上被安排进入政治局常委会后，分管中共中央组织工作，刚满一年，陈希即升任国务院教育部副部长兼党组副书记。在此位置上不足两年时，习近平又于2010年9月安排陈希外放辽宁，以省委副书记职务为晋升正省部级热身。2011年4月，也就是陈希在辽宁省委副书记位置上满打满算才坐了7个月的时间，习近平又将陈希调回京城，接替邓小平次女邓楠的中国科协党组书记、副主席和书记处第一书记职务。
  <br/>
  <br/>
  中共十八大上，习近平上台出任中共总书记，陈希亦步步高升。上台之初，习把自己的人事总管职务（中组部常务副部长）交给自己“铁哥们”陈希，同时也为他预设好了副国级待遇的未来。中共十九大上，陈希进一步高升，同时担任中央政治局委员、中央书记处书记，中央组织部部长，中央党校（国家行政学院）校长（院长）等要职。
