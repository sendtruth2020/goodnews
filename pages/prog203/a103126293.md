        <ok href="https://www.ntdtv.com/gb/拉丁美洲.htm">
         拉丁美洲
        </ok>

      </span>
     </span>
    </span>
   </span>

   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        美国总统拜登（Joe Biden）已承诺要向海外分发8000万剂疫苗，但尚未宣布他将向哪些国家提供捐赠。
       </span>
      </span>
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        国会议员和地区专家们说，美国政府需要赶上中共，向拉丁美洲派送疫苗，并确保全世界都知道这种疫苗来自美国。
       </span>
      </span>
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        根据泛美卫生组织（PAHO）的数据，包括智利、萨尔瓦多、巴西和乌拉圭在内的几个拉丁美洲国家几乎完全依靠中国制造的疫苗。
       </span>
      </span>
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        俄罗斯也向该地区发送了较少数量的疫苗。
       </span>
      </span>
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        洪都拉斯和巴拉圭面临着疫苗短缺的局面，但尚未接受任何中国制造的疫苗。
       </span>
      </span>
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        两国都表示，中共允诺向它们提供疫苗，但是要求它们断绝与台湾的外交关系。
       </span>
      </span>
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        大西洋理事会拉丁美洲中心主任杰森·马尔扎克（Jason Marczak）说：“中国利用这一时刻在该地区发挥了作用。”
       </span>
      </span>
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      尽管中共隐瞒疫情造成了中共病毒大流行，但是中共一直致力于将公众的注意力集中在它如何帮助其他国家抗击病毒上。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      马尔扎克说，中共试图将自己作为中共病毒问题的中心转变为中共病毒解决方案的中心。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      洪都拉斯高级官员卡洛斯·阿尔贝托·马德罗（Carlos Alberto Madero）最近表示，对疫苗的需求使他的国家“处于非常困难的境地”，他不排除必须与台湾断绝关系。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      马德罗告诉伦敦《金融时报》，洪都拉斯人看到中共正在发送疫苗，“我们开始问，为什么我们的盟友没有帮助我们”。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      除了向加拿大和墨西哥捐赠约400万剂疫苗外，拜登政府官员仍在考虑向哪些国家派送疫苗。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      议员和地区专家正在推动白宫将拉美和加勒比地区作为优先地区，他们认为无论从公共卫生和战略角度来看，这样做都是合理的。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      “没有美国的参与和领导，我们的竞争对手将继续努力使用效果不佳的疫苗作为手段来胁迫拉丁美洲和加勒比海国家，以支持对我们不利的外交议程。”三名参议员——共和党人卢比奥（Marco Rubio），民主党人梅嫩德斯（Bob Menendez）和凯恩（Tim Kaine）上周在致拜登的信中说。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      拜登向其他国家捐赠8000万剂的承诺远远超过了其他政府的承诺。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      华盛顿已经承诺提供40亿美元，以支持由世界卫生组织（WHO）支持的国际COVAX平台，该平台向有需要的国家捐赠疫苗。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      国家安全委员会发言人说：“我们将与COVAX和其他合作伙伴合作，确保以公平的方式提供安全有效的疫苗，并遵循科学和公共卫生数据。”
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      发言人表示，“重要的是，我们的疫苗没有附带条件”。 他补充说：“我们与世界共享疫苗，并在全球疫苗战略中处于世界领先地位，因为这是正确的做法：道义上正确，从全球公共卫生角度来看正确，对我们的集体安全和福祉也正确。”
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      至于洪都拉斯，发言人说，该国从COVAX那里收到的疫苗数量比其他任何来源都多，并且将在未来几周内通过COVAX平台获得更多疫苗。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      美国陆军战争学院战略研究所的拉美事务研究教授埃文·埃利斯（R. Evan Ellis）表示，在拉丁美洲，北京主要是向该地区的国家出售疫苗，而不是捐赠。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      埃利斯说，尽管如此，中共在社交媒体和官方媒体中都将自己描绘成在拉丁美洲国家面临致命性大流行病时向其提供帮助。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      埃利斯周四对美中经济与安全审查委员会表示，美国的疫苗捐赠使“中国人所做的一切黯然失色，但中国人已将每次在机场停机坪上的交付都变成了拍照机会”。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      众议院外交委员会首席共和党议员麦考尔（Michael McCaul）表示：“我们不允许中国共产党利用这一大流行病破坏我们在拉丁美洲和加勒比海地区的国家安全利益。” 他说，洪都拉斯等国家承受着断绝跟台湾外交关系的压力，以便从中共获得急需的疫苗。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      在中共病毒大流行初期，中共采用“口罩外交”来提高其在拉丁美洲地区的知名度，提供外科口罩和其他医疗用品。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      在过去的十年中，中国已在拉丁美洲市场上取得了重大进展，超过了美国，成为巴西、智利、秘鲁和乌拉圭的最大贸易伙伴。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      中共在拉丁美洲投资港口，道路，水坝和铁路，经常向拉丁美洲政府提供贷款，并大量购买矿产和农产品。
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      中共利用其经济实力来让华为和其他中共国有企业在拉丁美洲的电信和航天领域发挥关键作用。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      在加勒比海地区，中共资助了安提瓜、牙买加、格林纳达、圣卢西亚和多米尼加这些小国的政府大楼、道路和板球场的建设项目，并在新的港口和度假村投资了数十亿美元。
     </span>
    </span>
   </span>
  </span>
 </p>
 <p>
  <span style="vertical-align: inherit;">
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        贸易带来了政治利益。
       </span>
      </span>
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        格林纳达和多米尼加与台湾断绝了关系。
       </span>
      </span>
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        自2017年以来，北京在拉丁美洲说服了三个国家放弃对台湾的外交承认：巴拿马，多米尼加和萨尔瓦多。
       </span>
      </span>
     </span>
    </span>
   </span>
   <span style="vertical-align: inherit;">
    <span style="vertical-align: inherit;">
     <span style="vertical-align: inherit;">
      <span style="vertical-align: inherit;">
       <span style="vertical-align: inherit;">
        但是像巴拉圭这样拒绝如此做的国家则被排除在中共的公共工程融资之外，并面临出口农产品的贸易壁垒。

      </span>
     </span>
    </span>
