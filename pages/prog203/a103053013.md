  <strong>
   要求：
  </strong>
  <br/>

  </strong>
  <br/>
  <strong>

  </strong>
  <br/>

  <strong>
   要求：
  </strong>
  <br/>
  有新闻写作经验，独立搜集素材的能力。有英文新闻翻译及写作经验佳。中英文非常精通。
 </p>

  </strong>
  <br/>
