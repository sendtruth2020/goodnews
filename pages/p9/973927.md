  【看中国2021年6月4日讯】根据消息人士透露，最近
  <strong>
   朝鲜
  </strong>
  军第二军团直属的
  <strong>
   通讯大队
  </strong>
  第一中队发生了一件机密档遗失事件，导致整个
  <strong>
   军团
  </strong>
  进入紧急状态。
  <br>
   <br>
    1日，DailyNK的
    <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C%E6%B0%91%E4%B8%BB%E4%B8%BB%E4%B9%89%E4%BA%BA%E6%B0%91%E5%85%B1%E5%92%8C%E5%9B%BD">
     朝鲜
    </span>
    内部部队消息人士指出，在5月22日下午，第二军团指挥部的直属通讯大队第一中队于进行换班交接之时，发生了一件“执勤日志”与“无线电密码”丢失事件。
    <br>
     <br>
      在无线电密码内有部队之间的无线电通讯暗号与频道等，因此军法中明确规定，在换班之时，执勤人员应按规定必须彻底交付好相关的秘密档。
      <br>
       <br>
        在事件发生之后，部队立刻找来换班前执勤的金某（女，18岁，列兵）进行调查，金某却相当坦白地表示，“撕碎且吞掉了文件”。
        <br>
         <br/>
         若未调查清楚此事件，有可能必须更换全军的通讯网无线电密码，而让第二军团面临了史无前例的严重局势，所以一定要严肃调查整起事情的原委。
         <br/>
         <br/>
         金某于接受第二军团保卫部第四部的审讯之时，她十分坦白的理由，颇让人大感意外，她声称主要是为了报复班长裴某，才会犯下此行为。
         <br/>
         <br/>
         最后调查的焦转移至裴某身上，调查结果发现，裴某掠夺金某的物品似乎已经成了惯性。

  <br/>

  经过调查后裴某的恶行就被揭露出来。她不只剥削了金某，也还夺取了其他士兵的物品，而拿去出售攒下了“转业基金”。

  <span href="https://www.secretchina.com/news/gb/tag/军团" target="_blank">
   军团
  </span>
  政治部首先以虐待下属为由，并损坏上下平等原则的嫌疑，已将裴某降级为普通士兵了。
  <br/>
  <br/>
  金某则因为将部队的机密档用为个人报复的手段，而自无线通讯兵部队调至高射炮中队。

  <br/>
  由于金某坦白的事情真实可信，所以部队就决定继续使用原来的无线电密码。
