 <p>
  【看中国2020年12月4日讯】澳中两国因为反映澳洲士兵战争罪行的
  <span href="https://www.secretchina.com/news/gb/tag/插画" target="_blank">
   插画
  </span>
  《和平之师》发生的外交纠纷还在发酵。原画创作者乌合麒麟日前发表了同一主题的另一幅画作《致莫里森》。
  <strong>
   中国
  </strong>
  外交部发言人华春莹又继续借题发挥。为什么西方国家对中国
  <strong>
   外交
  </strong>
  官用这些
  <strong>
   插画
  </strong>
  大做文章感到不满呢？
  <br>
   <br>
    乌合麒麟的电脑插画《和平之师》，取材于真实的事件，描述了澳洲士兵在阿富汗战场上把沾满鲜血的匕首放在一位平民小孩脖子上的场景，而配文却是“别害怕，我们是给你带来和平的。”
    <span id="hideid" name="hideid" style="color:red;display:none;">
     <span href="https://www.secretchina.com">
     </span>
    </span>
   </br>
  </br>
 </p>
 <p>
  <strong>
   问题在政府
  </strong>
  <br>
   <br>
    画作发表后，
    <span href="https://www.secretchina.com/news/gb/tag/中国" target="_blank">
     中国
    </span>
    外交部发言人赵立坚和华春莹先后利用这幅画批评澳洲政府的人权纪录，并且拒绝了澳洲总理莫里森要中国方面道歉的要求，引发国际社会议论纷纷。法国、新西兰和美国政府先后公开谴责了中国方面的做法。
    <br>
     <br>
      华盛顿智库战略与国际研究中心高级研究员约翰·绍斯（John Schaus）认为，画家个人以画作来批评政府，固然是他个人的言论自由，但中国政府把这种画用在正式场合却有违国际准则。
      <br>
       <br/>
       “中国外交部发言人的做法，对澳洲政府及其人民，既不尊重，也不恰当，因为他们是用非真实的图像来论证他们的观点。”
       <br/>
       <br/>
       他强调，政府的正确做法应该是陈述事实，而不是扭曲。
       <br/>
       <br/>
       但中国政府的做法并没有止步于此。在中国政府用非真实的图像指责澳大利亚的同时，由中国官方控制的社媒平台微信又封锁了澳洲总理莫里森在微信上发表的相关言论。
       <br/>
       <br/>
       据路透社报道，莫里森的微信发言批评了中国政府使用非真实图像（乌合麒麟的插画中的澳洲士兵是用的照片），也表示会用透明的方式处理澳洲士兵的战争罪行。
       <br/>
       <br/>
       <strong>
        中国的这一系列做法引来了民间的强烈反弹。
       </strong>
       <br/>
       <br/>
       澳洲《每日电讯报》日前在头版刊登了标志着1989年天安门屠杀事件的“坦克人”照片，并且仿效赵立坚的口吻注释说，“我们对中国平民和学生被中国士兵屠杀感到震惊。我们强烈谴责这样的行径，并呼吁向他们追责。”末尾还捎带一句：这是真实的照片。
       <br/>
       <br/>
       学运领袖王丹也把这一事件与天安门屠杀相比较，“中国当然可以找到美国等西方国家一系列的人权问题，但这些问题都是这些国家自己暴露出来，而且允许监督和批评的。而恰恰是中国自己有那么多问题，完全地压抑，1989年六四事件过去三十多年了，还是不许讨论。”
       <br/>
       <br/>
       <strong>
        问题也在个人
       </strong>
       <br/>
       <br/>
       但中国外交部发言人华春莹似乎并没有理解国际社会对他们做法的不满。在12月2日的例行记者会上，她延续了日前在谈及《和平之师》时对澳大利亚的谴责态度：
       <br/>
       <br/>
       “他真正恼怒的是，中国人怎么有这个权利，去对他的坏事，把它指出来，或者发表评论。”
       <br/>
       <br/>
       而自称“战狼画家”的乌合麒麟也不甘寂寞，继《和平之师》后，乌合麒麟又于日前发表新作《致莫里森》，把矛头直接指向了澳洲总理莫里森。
       <br/>
       <br/>
       乌合麒麟以画自况，指责莫里森和西方媒体试图掩盖澳大利亚士兵的战争罪行，或者视而不见，却把矛头对准画家自己。
       <br/>
       <br/>
       这种立场与中国官方不谋而合。华春莹在12月2日的例行记者会上，继续谴责说，“我们希望，澳方以及美方的一些人，能够秉持客观、理性的态度，能够正确地认识自己，正确地认识别人，能够有自知之明，也能有知人之明。”
       <br/>
       <br/>
       但乌合麒麟的新作《致莫里森》已经完全偏离了事实。
       <br/>
       <br/>
       事实是，对澳洲士兵战争罪行的调查结果《布里尔顿报告》（Brereton report）已经在澳洲国防部的官方网站上公布。澳洲国防军司令坎贝尔（Angus Campbell）在报告发布时，就为澳洲士兵在阿富汗的行为给予了道歉，并表示将依法处理。
       <br/>
       <br/>
       <strong>
        中国对外关系的新低
       </strong>
       <br/>
       <br/>
       美国国务院在对中国政府针对澳洲的这些做法提出谴责时说，这是中国在散布虚假信息方面到达了新的下限。
       <br/>
       <br/>
       美国这么说，又显然牵系着中国政府在新冠疫情期间散布假信息的行为。
       <br/>
       <br/>
       约翰·绍斯认为，中国的做法将对澳中关系造成进一步的破坏，“澳洲民众自然会认为，这是中国分裂两国关系的行为，中国将失去他们在澳洲民众心目中的地位。这种沟通策略必将会导致反击。这就跟新冠疫情期间，中国在欧洲使用战狼外交导致的结果一样。”
       <br/>
       <br/>
       王丹也认为，这种战狼外交尤其破坏中国在外国民众心中的形象，“尤其会引起民间的反感，最近的多项民调表明，世界多个国家老百姓对中国的反感都有所增加，这些增加都是由中国政府的这些做法导致的。”
       <br/>
       <br/>
       中国政府的做法似乎和他们的说法并不自洽。就在两天前，中国外交政策负责人之一杨洁篪在官媒《人民日报》发表文章说，在未来的外交工作中，要树立负责任大国形象，并积极营造良好外部环境。
