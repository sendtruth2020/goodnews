 <p>
  【看中国2020年12月31日讯】目前
  <strong>
   朝鲜
  </strong>
  在
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
    武汉肺炎
   </span>
  </strong>
  的冲击下，人民生活更加困苦。该国第八次党代会临近，为了防范此疫情而加大防疫力度，但百姓们对于这种政策的反应已经越来越冷淡。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  根据消息人士表示，朝鲜因为过分的防疫措施，导致经济情况越来越严峻，民心亦越来越冷淡了。尤其有越来越多的干部认为这种方针仅是为了躲避今后的责任而已。
  <br>
   <br>
    28日，平壤消息人士向DailyNK说，由于制裁、严重的传染病病毒，以及自然灾害等因素，已面临经济危机。由于防疫措施，使得市场流通与百姓移动受到限制，致使赚一天过一天的人们生计遭到严重打击，对于当局的怨声也越来越大。
    <br>
     <br>
      平壤政府依据防疫阶段及各地武汉肺炎扩散的情况，采取了不同力度与局部性的限制人员和物资的移动措施。
      <br>
       <br>
        自去年12月初开始，朝鲜政府作为
        <span href="https://zh.wikipedia.org/wiki/2019%E5%86%A0%E7%8A%B6%E7%97%85%E6%AF%92%E7%97%85">
         武汉肺炎
        </span>
        “超特级”防疫措施的一环，一直至明年2月底为止，禁止没有经许可的人员与物资的地区间移动。
        <br/>
        <br/>
        尽管在表面上与以往的措施相似，可是针对武汉肺炎防疫而进行封锁以来，已经于各地出现了经济萧条现象的情况下，当局想试图强调防疫措施，以笼络民心。
        <br/>
        <br/>
        消息人士认为，各地都说目前的经济情况，已经比苦难的行军（也就是1990年代中后期的大饥荒时期）更加严峻。所以平壤政府认为必须收拾好这次事态。
        <br/>
        <br/>
        但据消息人士说，连干部们亦皆认为金正恩政权想特别强调现今的经济危机是因为全世界流行病毒所导致的。
        <br/>
        <br/>
        虽然朝鲜当局已通过加强宣传，以消除针对政府的不满情绪，而试图团结民心，但这并不容易。
        <br/>
        <br/>
        据说，也有人指称，政府加强武汉肺炎防疫措施，并不是实质性的对策。亦即政府为了避免年政治危机，而想采取“武汉肺炎政治”。
        <br/>
        <br/>
        还有人说，这是试图将经济失败的责任，转嫁给部分干部身上的一种手段。亦即政府的武汉肺炎应对防疫政策是优秀的，但只因一些干部的无能与腐败，才会变得更加严重。
        <br/>
        <br/>
        事实上，一直以来，朝鲜当局都在采取处罚责任人的方式，以转移百姓对于政府严重政策失误的不满情绪。去年9月，因为台风咸镜道遭到严重灾害之时，金正恩国务委员长就撤掉咸镜南道党委员长金星日之职务。2010年，当时的劳动党计划财政部部长，也因为货币改革失败，遭到枪毙。
        <br/>
        <br/>
        朝鲜的武汉肺炎防疫措施，已经越来越不理性了。有高层干部因为走私而被枪毙，甚至还张贴布告称，如果靠近边境地区，将会进行射击等。在不久之前，还曾因为有一位居民偷渡回朝鲜，当局以此为由，封锁了整个三池渊市。
