 <p>
  【看中国2021年3月13日讯】
  <strong>
   乌克兰
  </strong>
  宣布，出于国家安全考虑，将在不久的将来把世界顶尖的飞机引擎制造商
  <strong>
   马达西奇
  </strong>
  公司重新国有化，以免其落入
  <strong>
   中国
  </strong>
  公司手里。
  <br>
   <br>
    <span href="https://zh.wikipedia.org/zh-cn/%E4%B9%8C%E5%85%8B%E5%85%B0" target="_blank">
     乌克兰
    </span>
    国家安全和国防委员会秘书达尼洛夫周四（3月11日）表示，“马达西奇公司将会通过符合法律和宪法的方式回到乌克兰人民的手中，回到乌克兰国家的手中。”
    <br>
     <br>
      达尼洛夫强调说：“这样做是为了我们国家的国家安全。”
      <br>
       <br>
        金融时报说，达尼洛夫领导的国防委员会已经通过了这个国有化方案，预计，乌克兰议会也会迅速予以批准。
        <br>
         <br/>
         乌克兰方面的这个决定可能会让企图收购马达西奇公司的中国公司北京信威科技集团旗下的北京天骄航空公司感到失望。北京天骄掌握着马达西奇公司的大部分股权。它数次向乌克兰反垄断委员会提出收购申请，但一直都没有获得批准。
         <br/>
         <br/>
         中国媒体称，中国投资方认为乌克兰政府的这个国有化决定是非法的，违反了市场经济规则。
         <br/>
         <br/>
         中国外交部发言人赵立坚在周五的记者会上表示，中方已经注意到这个消息，“中方要求乌方依法维护中方企业和投资者的合法权益，妥善处理相关问题。”
         <br/>
         <br/>
         中国投资方收购马达西奇公司的目的是获取这家公司的飞机发动机制造技术和知识产权，从而推动中国的军事发展，特别是飞机发动机、卫星与巡航导弹的制造。美国认为，这将直接威胁到美国的国家安全，因此在北京天骄收购马达西奇的交易上持坚决反对的立场。
         <br/>
         <br/>
         在乌克兰政府作出这个决定之前，美国宣布对北京天骄进行制裁。美国制裁这家中国公司的理由是，它与中国人民解放军关系密切。
         <br/>
         <br/>
         美国国务院在1月14日发表的一份声明中说，北京天骄在乌克兰进行掠夺性的投资和技术收购构成了“不可接受的风险”，它有可能把这些技术用于中国的军事用途。
         <br/>
         <br/>
         今年1月底，乌克兰总统泽连斯基签署命令，对北京天骄等4家中国公司，以及3名中国公民实施制裁，包括冻结资产、限制贸易、限制入境、限制撤资与增资、限制证券交易等。
         <br/>
         <br/>
         马达西奇是在上个世纪90年代实行私有化的，公司的股份卖给了公司员工，其中很大一部分股份逐步落入外国公司之手。
         <br/>
         <br/>
         在多次收购受阻之后，北京天骄扬言要进行报复。达尼洛夫周四表示，乌克兰方面不会屈服于法律威胁。丹尼洛夫表示，那些投资于这家工厂而不是外国公司的人，在国有化之后将会得到一定的补偿。马达西奇公司的一少部分股份还会在市场上交易。
