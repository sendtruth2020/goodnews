  【看中国2020年11月20日讯】
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
    武汉肺炎
   </span>
  </strong>
  仍在全球蔓延着，但
  <strong>
   朝鲜
  </strong>
  迄今还宣称零确诊，对此，外界却持怀疑态度。据了解，因发生了武汉肺炎疑似症状而遭到封锁的朝鲜慈江道万浦市，已经于上周末解除封锁。在封锁期间万浦市有100多名百姓死亡，因而人心依然惶惶。
  <br>
   <br>
    17日，慈江道消息人士向DailyNK透露，在14日解除了万浦市的传染病防疫封锁。于封锁期间民众不能够上班，仅能留在家中，现今能够上班了，也可以出去逛逛。
    <br>
     <br>
      先前，DailyNK引述消息人士的话报导，由于万浦市发生了10多名武汉肺炎疑似症状的患者死亡，当局立刻在10月26日采行封锁措施。据悉，死亡人员皆曾经接触过走私进来的物品，因此朝鲜政府就进行紧急消毒，以及将接触人员隔离等。
      <br>
       <br>
        据消息人士透露，在封锁万浦市之后，当局也烧毁了出现问题的全部走私品，有部分重要的物品，仍然还存放于万浦海关仓库内，并每天进行消毒。
        <br/>
        <br/>
        据了解，没有遵守隔离存放海外引进物品，过一段时间后进行处理措施的万浦市海关工作人员和商贩等10多名人员，已经在万浦市安全部拘留所接受调查。
        <br/>
        <br/>
        有分析认为，他们已经涉嫌试图颠覆国家罪及杀人罪，最少会遭判处7年劳动教化，最长可判处无期徒刑，甚至其中有一两名主犯，还可能会被公开处决。
        <br/>
        <br/>
        据说，由上报道紧急防疫委员会的资料中显示，在封锁期间万浦市因为武汉肺炎疑似症状死亡人数为107名，以及有320名被隔离。
        <br/>
        <br/>
        据消息人士说，需要进一步确认的人员，他们在家进行自我隔离，而市安全部会在这些家庭门上，贴着“隔离”字样，以进行管控，并防范一般百姓靠近。
        <br/>
        <br/>
        目前尽管情况依然相当严峻，但朝鲜当局认为有必要让百姓回归到正常生活，因此才解除了封锁。
        <br/>
        <br/>
        消息人士称，万浦的冬季长，必须烧柴至第二年的五月初。但当局在最重要的筹备柴火期间实施禁止移动，因此人们都无法做过冬的准备，有一半以上的家庭，尚无腌过冬用的泡菜。若再继续封锁下去，则将没有活路了，就只好解除封锁。
        <br/>
        <br/>
        平壤政府封锁万浦市之后，人均每日300克的标准，供应了10天份的粮食，但这远远就不够支撑3周的封锁。据悉在万浦市幼稚园与小学中，患有营养不良的10多位孩子，竟然因粮食紧缺而饿死了。

  消息人士表示，道和市里民众说，自从解除封锁的14日起再观察20天。仅能运入军备工厂所需的物资，但不能送来人们生活所需的大米等，因此市场都没有货了。
