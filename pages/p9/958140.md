 <p>
  【看中国2021年1月5日讯】先前
  <strong>
   朝鲜
  </strong>
  强调社会主义优越条件之时，总是会提及“无偿医疗制度”。但随着
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
    武汉肺炎
   </span>
  </strong>
  的影响扩大，该国的免费医疗制度，完全可称是一个“有名无实”的制度了。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  2020年3月，平壤综合医院竣工仪式后，
  <span href="https://www.secretchina.com/news/gb/tag/朝鲜" target="_blank">
   朝鲜
  </span>
  中央电视台等朝媒曾经报导称，在平壤综合医院竣工之后，人民将会在现代化医院享受无偿治疗，届时国际社会都十分羡慕我们有优越的社会主义保健制度。
  <br>
   <br>
    然而，随着加强武汉肺炎防疫措施以来，平壤综合医院竣已经一拖再拖，甚至连在隔离设施的百姓，也无法获得药品。
    <br>
     <br>
      据说，一些有钱人必须自费购买药品来服用。多名消息人士透露，如果生命垂危的重症患者，不花钱买药就无法得到治疗。如同上所述，其实，朝鲜的无偿医疗制度早就已经有名无实了，而由于武汉肺炎疫情使其更进一步陷入深渊。
      <br>
       <br>
        <strong>
         目前朝鲜的无偿医疗服务，仅是进行“诊断”而已
        </strong>
        <br>
         <br/>
         据Daily NK通过采访得知，朝鲜的无偿医疗服务现状，免费的只有医生的诊断。在诊断之后并没有提供药品与注射等措施。
         <br/>
         <br/>
         若从市场或通过个人的管道拿来药品，医院就会替其注射。但这也必须向医院缴交服务费才可以。
         <br/>
         <br/>
         4日，有消息人士向DailyNK表示，开始苦难行军的90年代中期起，就已经无粮食供应，连免费治疗也都没有了。现今每人都要赚钱，而且没有一个医生不收治疗费用及手术费等费用的。
         <br/>
         <br/>
         据了解，医疗服务费可分为手术费和治疗费，并依据地区和医院的等级来收取费用。同时也有一些地方还必须单独支付给医生辛苦费。
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  <strong>
   武汉肺炎医药品紧缺 各地出现短货
  </strong>
 </p>
 <p>
  朝鲜在武汉肺炎出现之后，发生进口药品剧减的情形，普通百姓已经很难于市场上买到药品了。在市场或各村药店能够买到常备的药品，但进口药品仅能通过走私商贩、医院及药局工作人员，以走后门的方式购买。
  <br/>
  <br/>
  在过去，能够买到退烧药、止咳药和止泻药等基本药品，以及注射针、注射器、消毒器与输液等。但现在不只很难弄到这些东西，而且价格也暴涨了。
  <br/>
  <br/>
  据消息人士披露，跟去年1月末封锁边境之前相比，进口药品价格已经涨了10倍以上，而且卖家要卖多少钱，就是多少钱。连平壤制药厂及顺川制药厂等朝鲜药厂所生产的药品价格，也上涨超过了6倍。
  <br/>
  <br/>
  据说，朝鲜的医疗机构针对一直发高烧与拉肚子的患者，都诊断为“副伤寒”，之后找退烧药及治疗消化不良的药品的人，也增多了，使得感冒药和止泻药等药品的价格，出现明显上涨了。
  <br/>
  <br/>
  <strong>
   朝鲜宣传的“
   <span href="https://www.secretchina.com/news/gb/tag/无偿医疗制度" target="_blank">
    无偿医疗制度
   </span>
   ”仅剩空壳
  </strong>
  <br/>
  <br/>
  在此种情况之下，平壤当局固守仅剩空壳的“无偿医疗制度”。尤其是武汉肺炎于国际大流行的情况下，朝鲜却宣称没有一名确诊患者，并宣传党与国家的保健医疗政策。亦即继续宣传其所谓的无偿治疗制度、党的预防医学制度，以及医生负责制等社会主义的三大保健医疗制度更优于资本主义国家。
 </p>
 <p>
  消息人士表示，若无偿治疗制度消失了，就是否定了领袖金日成的社会主义革命历史。此即是不能够随意改善保健医疗制度的一个主要原因。因此，朝鲜的保健医疗制度，并非给人民提供一个实际的医疗服务的医学体系，而是当局宣传成果的政治宣传手段。
  <br/>
  <br/>
  在此种情况下，当武汉肺炎疫情越来越严重，就转为宣传该国没有武汉肺炎患者，以此来宣传政府的优越。
