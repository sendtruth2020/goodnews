 <p>
  【看中国2020年12月16日讯】
  <span href="https://www.secretchina.com/news/gb/tag/朝鲜" target="_blank">
   朝鲜
  </span>
  在领导人金氏家族的统治之下，成为一个极为封闭的国家，深怕外界资讯传入，以致动摇其地位。据悉，最近
  <strong>
   朝鲜
  </strong>
  举行了最高人民会议常任委员会全员会议，并通过拒绝反动思想文化法，以禁止引进与流通各种的国外
  <strong>
   新闻
  </strong>
  、外国文化内容等国外资讯，且严谨体制
  <strong>
   偶像化宣传物
  </strong>
  等内部的资讯外泄。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  据DailyNK通过采访获知，在拒绝反动思想文化中，规定禁止收听、录音和流通国外广播；如果引进及流通国外的不纯影像制品、图书与出版物、复制和流通没有经过国家允许的音乐等行径，都是属于非法行为。
  <br>
   <br>
    据了解其中以收听海外广播的问题，排在第一位。亦即平壤政府对于不特定多数人收听的广播尤其敏感。
    <br>
     <br>
      所谓“禁止国外影像制品”，主要针对的是韩国文化方面的内容。在
      <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
       武汉肺炎
      </span>
      疫情而封锁边境之后，朝鲜观看韩国电视剧或电影的百姓似乎增多了。
      <br>
       <br>
        而外国出版物中也包括了圣经，由此可见日后将会加强宗教方面的镇压。
        <br>
         <br/>
         此外，还明确规定多人观看不纯影像制品、图书而被发现，主谋者将会受到公开审判，同谋也将遭法律惩罚。
         <br/>
         <br/>
         同时还规定引进没有允许的影像制品或图书的人，将会受到公开审判。在结束相关刑法的修订后，亦会增加公开进行审判的频度，以加强“恐怖政治”。
         <br/>
         <br/>
         在拒绝反动思想文化法之中，更明确规定严控各种播放机内容，且将会无条件没收无张贴国家技术检查表的电子仪器。
         <br/>
         <br/>
         这里所称的无国家技术检查表的电子仪器，是指没有向国家保卫省中央第27局申报的电脑、手机、电视及影像播放机等。试图以此来防范国外资讯通过没有经过申报的仪器扩散出来。
         <br/>
         <br/>
         为此，在年初之时朝鲜政府就已经下令加强电子仪器的管理工作，以防范资讯和思想文化的引进及流出。
         <br/>
         <br/>
         若个人的手机中，储存没有经过国家允许的音乐，图片、非法打印图片或图书、电子图书等，都将会受到法律的处罚，以严格打击流通的各种资讯内容之行为。
         <br/>
         <br/>
         另一方面，更明确规定，将依法处罚边境沿线地区的走私、销售和流通无经国家允许的电子产品、人民生活消费品和杂货等行为。
         <br/>
         <br/>
         没有经过国家允许的工业产品，主要是指韩国生产的物品，因为朝鲜当局一直都严控韩国产品。政府认为当百姓使用了高品质的韩国产品，将会对韩国抱有幻想。
         <br/>
         <br/>
         在拒绝反动思想文化法之中，最为引人注意的是，明确地规定禁止复印、储存和散布“不灭的向导”、“不灭的历史”及“不灭的旅程”等金氏家族一家的偶像化图书。
         <br/>
         <br/>
         过去，朝鲜政府禁止携带国内图书至国外。但这次更禁止了用手机储存盘，欲借此来事先防范偶像化图书的流出。据了解，预计于明年1月下旬所举行的最高人民会议常任委员会上，将修订与拒绝反动思想文化法相关的具体处罚规定之刑法。
