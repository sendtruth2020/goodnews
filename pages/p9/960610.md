  </strong>
  疫情肆虐全球，
  <strong>
   世界卫生组织
  </strong>
  （WHO）于27日表示，在英国发现的
  <strong>
   变种病毒
  </strong>
  已经扩散至70个国家了，过去一天就新增逾1.8万人死亡，再创纪录新高。但
  <strong>
   朝鲜
  </strong>
  迄今仍对外宣称境内零确诊病例，外界对此存高怀疑态度。据消息人士透露，朝鲜政府在一些地区，并不隔离武肺疑似症患者，亦不进行积极治疗。所以当地百姓皆都自嘲称，武汉肺炎防疫也是自力更生。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  22日，一名咸镜北道消息人士向DailyNK表示，从15日开始，会宁市就不再隔离
  <span href="https://zh.wikipedia.org/wiki/2019%E5%86%A0%E7%8A%B6%E7%97%85%E6%AF%92%E7%97%85">
   武汉肺炎
  </span>
  疑似症患者，甚至还停止药品供应。现今只有确认发烧情况，而放弃了治疗。
  <br>
   <br>
    在朝鲜总共有9个武汉肺炎疑似患者的隔离设施。即是黄海道、平安道、咸镜北道、咸镜南道、江原道、两江道、慈江道、罗先市和南浦市等地区各有一个设施。
    <br>
     <br>
      据消息人士披露，其实这些隔离设施只是隔离而已，并不进行诊断或治疗。但在新年之后也不再进行隔离了，武汉肺炎似乎也成为一种“自力更生”。
      <br>
       <br>
        据消息人士指出，从本月中旬起，为了开展防疫工作，在上午9点30分至10点，下午3点至5点，每个医生都会在自己负责的地方，挨家挨户地进行2次查访。若发现有发高烧者，就会报告给地区医院及防疫所。
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  医生们仅进行“确认”，并不将高烧患者送到隔离设施，而只要求患者进行自我隔离。虽然有开了感冒药与解热镇痛类药品的处方，可是并不提供药品。
  <br>
   <br>
    因此，导致市场上药品价格的上涨。原先只有100朝元的感冒药，目前已经暴涨到了600朝元了；而原本仅80朝元的解热镇痛药，也跟着涨到了400朝元。
    <br/>
    <br/>
    对此，百姓们皆认为当局的这种“半放弃式”的防疫措施，简直是荒唐透顶。人们都说，医生什么都不做，那么有必要进行户访吗？让我们自己买药吃，如果这样就别再干涉了。
    <br/>
    <br/>
    消息人士表示，百姓们都以为在党代会结束之后，生活应该会变好，可是连武汉肺炎都还要自行来治疗，所以觉得很泄气。也有人气愤地说，既然都这样了，还不如解除边境封锁，好让百姓能够自行购买武汉肺炎药品。
    <br/>
    <br/>
    消息人士进一步表示，就像党代会所提出的那样，今年无论是生计或是武汉肺炎的防疫方面，都要靠自力更生，但却被限制的寸步难移，要如何进行自力更生啊？
    <br/>
    <br/>
    5日至7日，在中央委员会工作总结上，
    <span href="https://www.secretchina.com/news/gb/tag/金正恩" target="_blank">
     金正恩
    </span>
    国务委员长特别提及，新的国家经济发展五年计划的基本核心，仍然还是自力更生及自给自足。希望不要受环境变化而影响开展有计划的、稳定的经济建设。
