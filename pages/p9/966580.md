  </strong>

  <strong>
   美国
  </strong>
  ，而后平壤政府决定与马来西亚断交等，使得两国关系陷入僵局。在此种情况之下，朝鲜政府还下令必须及时报告海外工作人员的一举一动。
  <span id="hideid" name="hideid" style="color:red;display:none;">

 <p>
  22日，根据DailyNK中国对朝消息人士透露，
  <span href="https://zh.wikipedia.org/wiki/%E6%9C%9D%E9%B2%9C_(%E7%A7%B0%E8%B0%93)">
   朝鲜
  </span>
  政府在21日对驻派中国等海外使领馆下达了指示，要求必须加强海外工作人员的监视和报告体系。
  <br>
   <br>
    先前大使馆或领事馆对于贸易公司负责人，每一周进行一次总结。但今后规定每周将举行两次以上，而且还必须随时报告关于贸易工作人员个人的一举一动和特别举动。
    <br>
     <br>
      在撤离驻马来西亚大使馆的情形下，将可能会加强滞留于马来西亚监视人员的工作范围与权力，同时也会加强相互监视与报告等举措。亦即朝鲜当局担忧马来西亚事态会影响到驻海外工作人员，以致发生思想动摇或引发流亡等情事。
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  据消息人士表示，政府在这次的指令中强调禁止对贸易人员提及与马来西亚有关的任何东西等，要求必须守口如瓶。尤其党中央更向外务省下达一道指令，即是严禁向亲朋好友与家人提到关于任何马来西亚的讯息。
  <br>
   <br>
    但据了解不只是滞留于海外的朝鲜百姓，甚至连平壤的干部们亦听到了与马来西亚断交的相关消息。
    <br>
     <br>
      据悉，听到这个消息的百姓们，都对于政府无法保护冒着危险替党工作的贸易人员，而表露出相当失望的情绪。
      <br/>
      <br/>
      有朝鲜内部消息人士说，朝鲜人民对于国人被抓至敌对的美国，是一件让人感到耻辱的事。原本以为只要我们拥有核武器，那么在国际社会上就能够搞外交与贸易等。然而，如今就算有了核武，也无法采取任何的磋商及措施，令人十分震惊。也就是说，本以为有了核武器之后，即使违反了对朝制裁措施，也不会遭到打击的，可是通过这次事件之后，才知道有了核武器也无法保护自己国民的事实。
      <br/>
      <br/>
      另一位消息人士称，其实被抓到美国的贸易人员，并非为了自己进口奢侈品啊。因为将最高尊严（指
      <span href="https://www.secretchina.com/news/gb/tag/金正恩" target="_blank">
       金正恩
      </span>
      国务委员长）当成了一个像太阳般伟大的人，因此才会冒着生命引进了党所要求的物品。但党却无法提供保护，那为何还要再继续做这种危险的事呢？
      <br/>
      <br/>
      在海外工作的贸易人员们看到了文哲明遭遣送至美国后，对于违反对朝制裁引进严禁物品一事，已经产生了厌倦，同时也对当局相当失望。
      <br/>
      <br/>
      中国的对朝消息人士认为，这次的马来西亚事件，一不小心将会动摇于海外引进违禁产品的贸易工作人员们的思想。当局最担忧的就是发生这种情况。
     </br>
    </br>
   </br>
  </br>
 </p>

  <strong>
   朝马急断交怕暴露这些事？
  </strong>
 </p>
 <p>
  19日，朝鲜外交部透过了国家通讯社“朝鲜中央通信社（KCNA）”发布声明称，因为不满
  <span href="https://www.secretchina.com/news/gb/tag/马来西亚" target="_blank">
   马来西亚
  </span>
  把“无辜的朝鲜公民文哲明”引渡到美国，而宣布与马国断绝外交关系。马国外交部也在当日回应并谴责朝鲜当局此项决定有违相互尊重的精神，且要求朝鲜驻马大使馆人员必须在48小时内离境。
 </p>
 <p>
  韩国《中央日报》援引前情报人员的说法指出，朝鲜利用于马来西亚设立空头公司等方式，以钻漏洞来规避制裁，除了提供禁运物品和
  <span href="https://www.secretchina.com/news/gb/tag/洗钱" target="_blank">
   洗钱
  </span>
  之外，也通过此途径搜集韩国相关情报，而文哲明就是其中一员。这次文哲明的被引渡，朝鲜当局相当担心其在东南亚一带规避制裁的管道，可能因此而暴露。因此，才强行与马来西亚断交。
