  【看中国2020年12月24日讯】
  <strong>
   朝鲜
  </strong>
  的
  <strong>
   人权
  </strong>
  黑幕，令人不寒而栗，因而常遭外界严厉批评
  <strong>
   。
  </strong>
  据悉，该国将一些违反
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
    武汉肺炎
   </span>
  </strong>
  防疫守则的百姓，送进了
  <strong>
   政治犯收容所
  </strong>
  等，并采取极度的恐怖政治。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  21日，朝鲜内部消息人士向DailyNK披露，
  <span href="https://zh.wikipedia.org/wiki/2019%E5%86%A0%E7%8A%B6%E7%97%85%E6%AF%92%E7%97%85">
   武汉肺炎
  </span>
  发生之后，如果没有遵守防疫规定的百姓、干部及军人们，都会烙上反抗政策的烙印。在这些人中若情节严重的和主谋者会被枪决，其他人员则由国家保卫省押送至政治犯收容所。
 </p>
 <p>
  <span href="https://www.secretchina.com/news/gb/tag/朝鲜" target="_blank">
   朝鲜
  </span>
  当局对于没有遵守国家紧急防疫规定的百姓们，直接扣上了危害国家经济及违反党政策的嫌疑，而将其当成特殊犯罪关入政治犯收容所。
  <br>
   <br>
    消息人士称，国家在早期封锁边境所采取的紧急防疫措施，被视为今年党的最高政策，还规定违反防疫措施的行径，为一种损毁国家的特大型犯罪行为。并试图造成违反武汉肺炎防疫者，并不具有公民资格的恐惧气氛。违反者是没有资格与党共命运的人，且进行彻底的处罚。
    <br>
     <br>
      <strong>
       在服刑期间停止所有公民权利
      </strong>
      <br>
       <br>
        消息人士表示，今年被送进政治犯管理所的人，皆为涉嫌违反防疫规定、违反国家贸易防疫管理，以及非法进出口国外物资。不只仅有党、政、军、社会团体、贸易人员和大学生等被关入管理所，而且连他们的家属，也有很多人遭关进去了。
        <br>
         <br>
          <strong>
           由于关押人员的剧增，而增设设施外及加派管理员
          </strong>
          <br/>
          <br/>
          据消息人士说，从平壤政府大力打击违反武汉肺炎防疫守则的人员以来，除了增设了政治犯收容所之外，也增派了人力。
          <br/>
          <br/>
          消息人士称，武汉肺炎刚发生之时，被送进管理所的人并不多。但到了夏天，关押到管理所的人突然增多了。
          <br/>
          <br/>
          华城、寿城管理所，以及人民保安省所管理的平安南道北仓、溪川管理所关押人员快速地增加了。为此还增设了一些牢房，并从保卫省、保安省下级人员及部队军官增派管理人员支援
          <strong>
           。
          </strong>
          也就是朝鲜政府从夏季开始，就针对违反防疫者加大了处罚的力度，因此收监人员也大幅增加了。
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  消息人士透露，5月至12月期间，光是华城、寿城政治犯管理所，就增派了自国家保卫省、部队下级军官调动过来的40多位的管理人员。而保安省管理所从4月至11月之间，也增加了60多名人员，且替换了以前人员的50%。
  <br/>
  <br/>
  <strong>
   收容所严重侵犯
   <span href="https://www.secretchina.com/news/gb/tag/人权" target="_blank">
    人权
   </span>
   ，已经成为死亡之地
  </strong>
  <br/>
  <br/>
  据了解，朝鲜政治犯收容所内的人权侵权程度非常严重，一直都没有得到改善。
  <br/>
  <br/>
  消息人士指出，5月，某一个管理所突然爆发了副伤寒，并死了一大批人。有人早上应该去干活儿的，但却不见出来，进去一看发现已死了。由于管理所不给药，因此囚犯都自然地认为若得病就会死亡。
  <br/>
  <br/>
  据消息人士说，收监人员一般都会因为营养失调、浮肿、皮肤病、劳动现场事故和处刑而死亡。
  <br/>
  <br/>
  消息人士进一步表示，如果尸体达到10具以上，就会一并焚烧。但并没有专门的火化炉，而在后山白菜地的旁边，或在仅剩草根的地方进行焚烧。
  <br/>
  <br/>
  由于当局担心副伤寒死亡者身上的病原菌会进行传播，就将尸体放在仓库内封起来。当尸体达到10具以上时，就会指使囚犯进入仓库，用消毒水来消毒之后，再搬出去焚烧。
  <br/>
  <br/>
  事实上，有一些关押设施没有专门的坟地，他们将尸体火化后，就不再管了。
  <br/>
  <br/>
  消息人士称，除了因为疾病或过度疲劳死亡者之外，也会公开来处决一些囚犯。
  <br/>
  <br/>
  消息人士更披露，11月，有某一个管理所当众吊死了一名囚犯。该尸体一直都吊在那里，过了一天后一看，已经被老鼠、乌鸦和昆虫等正在啃食呢。
