 <p>
  【看中国2021年1月15日讯】据悉，
  <strong>
   朝鲜
  </strong>
  为了防范
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
    武汉肺炎
   </span>
  </strong>
  而采取的禁止地区间移动的措施，其副作用十分严重。有消息人士指出，朝鲜各地出现了粮食严重不足的情形，甚至更发生了一些百姓为了取得
  <strong>
   粮
  </strong>
  食，而在
  <strong>
   边境地区
  </strong>
  徘徊，惨遭枪击的事件。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  13日，平安北道消息人士向DailyNK披露，目前做生意或者为了购买粮食外出的人，也都不能出门了。因此，粮食严重不足的家庭更加剧增了。
  <br>
   <br>
    从上个月开始，朝鲜已经将防疫措施提高至超特级紧急防疫阶段，除了部分物资与人员之外，其他都禁止移动了。
    <br>
     <br>
      消息人士称，在新义州等市内，虽然难一点但还能弄到吃的，可是在地方就不一样了。应该让人们能够过活，但却禁止移动，以致就出现了挨饿的百姓。亦即虽然新义州等大城市采取了严控移动措施，但因为有商店与流通等基础设施，因此能够搞到粮食，可是农村的情形就非常不如意了。
      <br>
       <br>
        由于去年遭到台风和洪灾等自然灾害，导致收成减少，又加上禁止人员及物资流动，以致没有基础设施的地方，就出现了粮食危机的情况。
        <br>
         <br>
          DailyNK也曾经报导过部分地区的百姓，为了取得粮食，而在深夜至农村搬运粮食的情形。为此，平壤当局还加强了严控措施。
          <br/>
          <br/>
          消息人士透露，其实，每天都能够听到有人饿死的传闻，但现在确实出现饿死的人了。虽然有美元的人，吃亏一点也会买得到粮食，可是情况较差的家庭，问题就非常严重。甚至还有人为了能够弄到粮食，而前往边境地区徘徊，但却遭到了枪击。
          <br/>
          <br/>
          消息人士说，有一些人认为至江边就能够弄点到什么，所以就到了江边，但却被打死了。为了生存而在江边徘徊中冤死，有很多人都觉得相当可怜。
          <br/>
          <br/>
          去年8月底，朝鲜张贴了一份布告称，如果靠近边境沿线1至2公里内，就会遭到开枪射击。因此，不断有消息说，警备队对着靠近边境的人或动物进行开枪射击的事，而且还听说非常频繁。
          <br/>
          <br/>
          据了解，百姓们对政府说境内没有武汉肺炎确诊者的宣传，已经持相当怀疑的态度了。
          <br/>
          <br/>
          消息人士表示，在进商店之前，有人会检查是否戴了口罩。当局既然说没有
          <span href="https://zh.wikipedia.org/wiki/2019%E5%86%A0%E7%8A%B6%E7%97%85%E6%AF%92%E7%97%85">
           武汉肺炎
          </span>
          ，为何连去卫生间，都还要求戴口罩呢？所以，人们认为这是政府在说谎。
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  根据世界卫生组织(WHO)在1月1日发表的各国武汉肺炎现状的报告中显示，
  <span href="https://www.secretchina.com/news/gb/tag/朝鲜" target="_blank">
   朝鲜
  </span>
  截至去年12月17日为止，针对了1万1707人进行2万3140次的武汉肺炎检查，但并没有发现一例确诊者。目前朝鲜政府还宣称该国境内没有发生一例武汉肺炎患者。
