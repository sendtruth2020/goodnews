 <p>
  【看中国2021年1月21日讯】去年11月，为了因应
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/武汉肺炎" target="_blank">
    武汉肺炎
   </span>
  </strong>
  疫情，
  <strong>
   朝鲜
  </strong>
  突然下令进行封锁，有一名仅能呆在家里的两江道百姓批指，只要市场正常运作，即使没有政府来帮助人们，也能生活得很好。但当局不想看到我们吃好喝好，因此以下令进行
  <strong>
   封锁
  </strong>
  而搞坏市场。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  最近，惠山市百姓金某接受了DailyNK的电话采访之时说，进行
  <span href="https://www.secretchina.com/news/gb/tag/封锁" target="_blank">
   封锁
  </span>
  之后，市场的物价涨得太高了，因此认为政府下令封锁，主要是为了搞垮市场。
 </p>
 <p>
  他披露，封锁令对于市场的影响很大。例如在惠山市，经封锁之后的食用油、砂糖和调味料等中国物品的价格，比封锁之前最少高涨逾两倍。由于封锁措施，使得人们无法做生意，消费亦明显减少了。
  <br>
   <br>
    因为种地的人们不知道何时会再进行封锁，以致不敢将农作物拿至市场上销售，而自行囤积起来。在封锁之后，市场柜台上的物品数量已经大幅减少，导致价格也随着增加了。
    <br>
     <br>
      去年8月，因为在中国居住的
      <span href="https://www.secretchina.com/news/gb/tag/朝鲜" target="_blank">
       朝鲜
      </span>
      百姓，通过三池渊而偷偷入境后，朝鲜当局就以防范武汉肺炎为由，而封锁了三池渊市与惠山市。
      <br>
       <br>
        平壤政府内部因为有一名自中国回来的人，就迳行封锁整个城市，这肯定有其他原因。当时，政府除了禁止三池渊与惠山市百姓移动之外，还全面中断了物资的运送。
        <br>
         <br/>
         去年11月初，当局再次封锁了惠山市。其起因是因为驻扎于两江道的边境警备队的保卫指导员，以及一位士兵趁着执勤展开走私的过程遭发现，最后竟然全副武装畏罪潜逃到了中国。
         <br/>
         <br/>
         而第二次封锁之时，朝鲜政府除了暂停惠山市所有工作人员上班之外，更禁止所有百姓出门20天。亦即采取了比第一次还要更为严厉的措施。
         <br/>
         <br/>
         由于惠山市与中国相邻，即使禁止移动，也还是不断发生走私事件，因此当局才会采取了比其他地方还要更为严厉的封锁措施。
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  金某表示，因为走私贩之故而封锁整个市是一个借口。声称没有武汉肺炎确诊病例，但却要采取此种措施，令人非常难理解。也就是说，有很多人都认为封锁并非为了防疫，而是出自于政治上的原因。
  <br/>
  <br/>
  由于民众不满情绪已经增大，在去年11月采取封锁之时，当局向每家每户供应了5公斤玉米，但后来因玉米不够了，而以土豆来代替。刚开始的时候，有传闻说实施封锁的时候，每家每户是供应10公斤的大米，但后来得知这并非无偿提供的，而是用以低于市场的价格来销售给百姓。
  <br/>
  <br/>
  百姓因为封锁而受到了相当大的影响。由于彻底禁止靠近鸭绿江，以致无法至江边洗衣服与打水了。金某说，刚开始前一两天的时候，只要有电就会在家看电视或打牌等方式来打发时间。但过了一段时间之后，却开始断水与断粮，可又不让人们出门，使得变得更痛苦了。
  <br/>
  <br/>
  在此种情况之下，前来协助封锁期间警戒的士兵，因为犯下暴行及偷窃等行径，更加重了百姓的痛苦。他们趁着百姓们无法出门的机会，有来自于其他地方的士兵，就到仓库来偷窃食品等，但当局并未对此情事要求索赔。
  <br/>
  <br/>
  尤其是无储备粮食或柴火的百姓们，已经出现了饿死或冻死的情形；同时也有必须急救的患者，因为没能获得及时的治疗，最后死亡了。金某称，还有为了生活而卖房子的人增多了，甚至在惠山洞的一家人，当他们卖了房子后，就一起吃了一顿丰盛的饭菜，之后竟吃老鼠药结束生命了。
  <br/>
  <br/>
  封锁之后不只经济方面受到严重打击，连社会氛围亦变得更加萧条，百姓对于当局的信任度，也下降了。金某指出，最近流行的是去掉“最高尊严”的“最高”这两个字，只称做“尊严”。人们嘲笑说，看样子，声称绝不会失败的尊严，用核武器是阻挡不了病毒的啊。
