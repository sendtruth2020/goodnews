  的慈江道慈城郡和万浦市，已经在17日解除了封锁。但有消息人士针对解除封锁令一事披露，在封锁间期，慈城与万浦竟然出现了100多名饿死者。

 <p>
  19日，慈江道消息人士向DailyNK透露，从17日零点起，解除了慈城与万浦的封锁令。在2月3日，慈城与万浦发生了
  <span href="https://www.secretchina.com/news/gb/tag/脱北" target="_blank">
   脱北
  </span>
  与走私事件，因而朝鲜当局对该地区发布了封锁令。原本计划于3月7日才解除封锁的，但只经过了2周就解除了封锁令。
  <br>
   <br>
    消息人士针对解除封锁令一事披露，在封锁期间期，慈城与万浦竟然出现了100多名饿死者。据说，尽管有一些因为饥饿而倒下去的百姓，他们送到医院后，也因没有葡萄糖液，而仅能于嘴内含着食盐水等。尤其是30多户军官家庭中，也出现了饿死者，所以这个问题就更加严重了。
    <br>
     <br>
      消息人士表示，当下达封锁令之后，军官不能够回家，仅能留在部队生活。经过一段时间之后，因为与家里联系不上，所以就打了电话给人民班长。当洞（街道）负责人至军官的家庭后，发现其家属已经倒在地上，就立即送至医院治疗，但还是去世了。
      <br>
       <br>
        消息人士说，今年1至2月，军官家庭只有拿到了军官的630克粮食供应，这只有10天份量的粮食而已，因此军官的家庭才会出现粮食紧缺的现象。但是百姓又不能够出门和做生意，使得每一个家庭的生活，就变得更加艰难了。
        <br>
         <br>
          他进一步表示，其实，早就没有东西吃了，以致才会发生这种悲剧。在事件发生之后，军官们就向政治委员提出抗议，且由道党书记向中央提交倡议书，因此才解除了封锁令。
          <br/>
          <br/>
          事实上，军官及其家属们都很不满地批指，我们又没做走私的是，只是忠自己的国家，可是为何死的是我们呢？当地的百姓也对当前生活危机，表示相当担心，并批判了当局的措施。
          <br/>
          <br/>
          由于百姓们十分不满，又加上军官家庭出现了饿死人现象，因此政府为防范发生动摇或骚乱情事，所以才提前解除了封锁。据悉，在解除了慈城与万浦封锁的同时，当局也向百姓们供应了粮食，此举是为了笼络民心所采取的措施。
          <br/>
          <br/>
          消息人士指称，17日虽然解除了封锁，并进行粮食供应，但供应数量粮食并不多。成人则按照1天400克，而儿童150克，以此来计算封锁期间供应玉米的量。
          <br/>
          <br/>
          消息人士称，约于15日，致使这次封锁的万浦市走私贩，以及协助走私的警备队士兵等共计7人，已经被送进了政治犯收容所，甚至其家属亦不知去向了。
         </br>
        </br>
       </br>
      </br>
     </br>
    </br>
   </br>

 </p>
 <p>
  先前消息人士表示，稳城与茂山等咸镜北道边境地区的物价不断地飙升，百姓们皆不知道要怎么办了，就连衣服扣也都天天涨价，已经涨了9倍之多。在此种情况之下，百姓饿着肚子也要勒紧腰带来节省费用。所以贫困的人仅能饿着肚子，甚至还发生了普通家庭一家人都饿死于路边的惨况。
 </p>

  消息人士说，去年11月24日有民众在茂山站前，发现了冻死于路边的一家4口。而安全部已经让郡机动巡逻队将尸体弄至别的地方了。
