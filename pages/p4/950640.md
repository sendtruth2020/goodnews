  进入十月以来，各种重磅事件精彩纷呈。在有限的时空里，积聚了太多高密度的信息，里面充斥着无数的垃圾碎片。现在距11月3日，只有十来天时间，大趋势已经确定，出现大变数的可能性越来越小。为了从复杂纷扰的表象中理出头绪来，我根据十月份发生的各种大事件，整理出四条意义重大的脉络，以供大家观察时参考。
  <br>
   <br>
    <strong>
     (1)
    </strong>
    <br>
     中共企图通过病毒攻击谋害川普总统、瘫痪美国政治、制造混乱帮民主党上台的如意算盘，已经落空。中共在10月1日下重手对川普总统发动病毒攻击，妄图一招致川普于死地。除川普总统、第一夫人外、总统团队的贴身幕僚多人同时感染，這种情況只能是遭到大剂量病毒的暗中定向攻击所致。令中共万万沒料到的是，川普总统的医疗团队釆用了新创疗法：瑞徳西韦（Remdesivir) 加上地塞米松（Dexamethason) ，将川普体内的病毒快速清零，使川普病情迅速转危为安，并在13天后捡测病毒转为阴性，还产生了扺抗＂中共病毒＂的抗体。中共的看似不可战胜的“新冠病毒”，在美国先进的医疗技术面前溃不成军，充分展现出美国科技力量的强大和深不可测。
     <br>
      <br>
       川普总统在不可思议的超短时间内康复出院，这只能是神的安排，体现着神的意志。左媒对川普神奇般的康复所表现出来的歇斯底里，正是对川普总统丶王者归来的礼赞。可以肯定，中共绝不会罢休，还会有新的攻击，但经过此劫，在受到神明护佑、大难不死的川普总统面前，中共的任何邪恶攻击都会失败。
       <br>
        <br>
         <strong>
          (2）
         </strong>
         <br>
          中共内斗中的“反噬”现象正在瓦解中共。凡是独裁体制，其内部都潜伏着独有的“反噬”机制。因为独裁体系的成员都是为个人的私利而结盟，为共同打劫分脏而入伙的。他们的“共同利益”，是建立在共同打劫分脏的基础上的，当外部环境安全，猎物资源充裕，各自划分的利益范围能够保障，分脏的流程和规则能够遵守时，他们是可以合作打劫的。而当上述条件消失了，他们的“共同利益”也就不存在了，没有了猎物，没有了围猎，没有了分脏，原来一致对外打劫的目光，就开始瞄向彼此的利益和财富了。于是，他们为了抢夺对方的利益(或者为了保护自己的利益)而互相争斗撕打，当强盗们开始互相打劫时，就表现出你死我活的“反噬”模式。这种“反噬”现象，在中共内部特别是统治高层，尤其表现得淋漓尽致和极其酷烈，每一笔财富的争夺，都赔付着鲜血和人命。
          <br/>
          <br/>

  10月6日，美国国务卿蓬佩奥、日本外相茂木敏充、澳大利亚外长佩恩和印度外长苏杰生在东京举行四方会谈，就如何应对中共在该地区咄咄逼人的扩张势头等问题进行磋商。几个覆盖了大半个地球的重要国家的外长，聚在一起专门讨论如何应对某国对该地区的威胁，这在前苏联垮台后近三十年来还是第一次。这标志着“印太小北约”的框架正式确立，以后其他国家再陆续加入。这个框架一确立，对抗、合围CCP的世界性联盟又打下一根桩基！
  <br/>

  与此同时，10月6日，台湾立法院推出“台美恢复邦交”、“请求美国协助抵抗中共”两项决议案，一读通过。重要的是，台湾立法院是由国民党主导，这两项议案的通过，意味着国民党主流派已经与马英九之流的亲共派彻底分道扬镳，并与蔡英文政府站入同一个战壕。它体现的是，台湾二千多万民众在抗击中共的威胁面前，已达成万众一心的空前团结。台湾的这条消息，与美、日、印、澳四国外长会议消息同一天发布，意义重大！

  <span href="https://www.secretchina.com/news/gb/tag/川普" target="_blank">

  (特朗普)连任已无悬念。美国民主党与邪恶中共长期勾兑的黑幕，正值美国民众因“新冠病毒”而对中共的厌恶感屡创历史新高的时候大量曝光，不仅大损民主党的百年声誉，而且将导致民主党输掉本次大选。拜登父子涉中共的腐败案所产生的冲击波，正在以乘数效应影响大选的走向，特别是享特硬盘里令人发指的虐童色情，不知会影响多少投向拜登的票转向川普。
  <br/>
  <br/>
  随着11月3日的临近，拜登父子的“中共门”事件还在加速发酵中，局面正在失控，越发不可收拾。然而，在享特硬盘里隐藏着一个天大的疑问却鲜有人追究：硬盘里的信息是从哪儿来的？先不说那数万封电邮，仅仅看这一千多幅照片和影像，场景都在中国，许多影像都是用针孔摄像头监控暗拍的，而非享特本人的自拍，因而这些照片影像肯定不是享特自己收集的，是由另一方提供的，这个“另一方”只能是中共。那么，问题来了，中共发这些足以定享特大罪的证据给亨特本人，其意图是什么呢？是为了娱乐吗？还是“学术交流”？显然都不是，中共的目的只有一个：胁迫、绑架享特为中共的利益服务，并间接影响和操纵拜登。
  <br/>
  <br/>
  假如，拜登手上也有一个录有他犯罪证据的硬盘，那这出戏可就大了去了！接下来，我们要追问的终极问题是：亨特硬盘的内容真的是“喝高了”不小心泄漏的吗？
  <br/>
  <br/>
  笔者不仿按逻辑常识作以下大胆推测：
  <br/>
  A、假如，中共要扶持上位的最理想的人选是贺锦丽(哈里斯)这个女人！但贺锦丽资历太浅，没人买账，而拜登是老政客，政治经验丰富，有一定人气，只是岁数太大，又被中共捏住了命根，是最理想的“面具”总统。所以，让贺与拜登搭档竞选是最优化的组合，如果拜登胜选当上总统，过渡一下，再以身体为由“病退”，贺锦丽再以副总统的第二顺位转正继任总统，如果那样，中共就完美实现了操纵美国大选的目标。的确，贺锦丽已经按捺不住自己的雄心，她在公开场合不止一次地表达过对总统职位的渴望！
  <br/>
  <br/>
  B、中共图谋让贺锦丽通过拜登之桥攀上总统大位，实际上是将拜登视为一碗“过桥米线”送到贺的嘴边。问题是，如果拜登父子知悉了中共的这一计划，他们会作何感想和选择？他们甘愿为她人做嫁衣吗？他们会听从党的安排坚决服从分配吗？非也！因为这是在美国。拜登父子确实在中共这里捞了巨额利益，但拜登在奥巴马时期对中共在美国的利益也给予了巨大的回报，他们与中共之间是互惠的利益关系，而不是主子与奴仆的关系。
  <br/>
  <br/>
  所以，中共想用黑材料绑架、操纵拜登父子听其摆布的企图，可能正好起到相反的效果。这就解释了为什么享特将电脑送修后“忘记了”取回？为什么电脑硬盘的内容曝光后拜登父子都保持沉默？为什么拜登团队甚至取消了竞选活动？为什么拜登的亲弟弟爆料直指贺锦丽等政客是拜登家族与中共华信能源合作的重要联系人？为什么享特的合作伙伴要大揭拜登家族与中共的非法交易？。。。。。。这些匪夷所思的问号，都共同指向一个巨大的可能——拜登父子难道会借自己的丑闻输掉选举，从而“炸掉”贺锦丽借道上位的拜登之桥？从而粉碎贺锦丽要当中共的傀儡总统的梦想？这个脑洞确实开的有点大，但近半年来曝光的事实证明，中共的邪恶远远超出我们开的脑洞。我宁愿把脑洞开得再大一点：拜登父子有性命之忧！
  <br/>
  <br/>
  目前来看，民主党已经黔驴技穷，大势已去。他们只剩下最后一招，就是选票做弊，但如果川普将竞选官司打到最高法院，最终还是川普胜选。民主党如果输掉了本次大选，不是川普打败了民主党，而是川普借中共之手打败了民主党。
