  <ok href="https://www.ntdtv.com/gb/广告.htm">

   误导
  </ok>
  小朋友。地铁方面回应表示，审核时考虑不周，将立即拆除。

 <p>
  爸爸，长大以后我想嫁给你！ 妈妈，长大以后我想娶你！这是贴在深圳地铁一号线车厢内的广告文案，主题是童言无忌。该广告文案引发网友不满，纷纷留言表示：
  <br/>
  “请不要
  <ok href="https://www.ntdtv.com/gb/误导.htm">
   误导
  </ok>
  孩童 出现这种标语良心何在 好变态”
  <br/>
  “这广告太离谱，违反伦理道德”、“没有底线吗？”
  <br/>
  “可以童言无忌，但是大人跟风是不是就是三观不正了”
  <br/>
  “虽然是童言无忌，但童言有很多啊，为什么是这些”
  <br/>
  “这种广告怎么批准的？”“什么人出得主意？”
  <br/>
  “这样的广告词也能一路通过审核，我也是醉了”
  <br/>
  “童言无忌就是伪命题。”
  <br/>
  “不要总拿孩子来做挡箭牌，关键在于这个广告不是给孩子看的！”
  <br/>
  “这明显是成年人做得文案，贴到大部分是成年人的环境里给成年人看，硬扯什么孩子嘴里说出来的话。”
 </p>
 <p>
  针对深圳地铁车厢内“童言无忌”广告语引发舆论风波一事，负责深圳地铁广告管理工作的深圳市地铁商业管理有限公司的一名工作人员回应表示，该文案由新东方深圳学校提供，于7月初开始放在车厢内，今天（8月5日）地铁停止营运后，将于明天（6日）凌晨撤下。
 </p>
