    拜登之子亨特可以说是“混”到极致，“硬盘门”中显示亨特不仅吸毒，还涉嫌与未成年人发生关系，这种罪行在美国是会被千夫所指，尤其是美国为代表的西方国家对未成年性犯罪可以说是零容忍，但是在左派媒体的保驾护航下，亨特不仅没有受到应有的惩罚，他的父亲还一路躺赢进了白宫，美国最强“官二代”非亨特拜登莫属。
    <br>
     <br>
      人不要脸天下无敌，逃脱了法律制裁的亨特不仅没有丝毫反省，反倒是大言不惭的出书自爆“黑历史”，其实就是犯罪史。亨特的新书《美好事物》中，他对与吸毒、制毒、酗酒、赌博、躲避药检、不伦恋等传言可谓是“供认不讳”。对于媒体的采访，拜登表示，他对于儿子敢于公开谈论成瘾一事感到“骄傲”，对于那些陷入人生困境者是个“启发”。
      <br>
       <br>
        真心尴尬，总统老子“护犊子”，是非不分，国家政权被其控制前途堪忧；儿子大言不惭，完美“坑爹”，多么完美的一对相爱相杀的父与子。不知道对美国民众内心作何感想，“护犊子”的爹加上败家儿子，满嘴仁义道德，背地里干的事情肮脏且龌龊。
        <br>
         <br>
          我们好好分享一下亨特是如何让拜登“骄傲”的，亨特拜登是一个十足的渣男，用大陆的网络梗来说他就是个“海王”。亨特和他罹患脑癌而不幸离世的哥哥博拜登相比就是两个极端，博拜登还曾是特拉华州总检察长，可惜博英年早逝。也许是最心爱儿子的离世让拜登不得不把所有的爱都转移到亨特拜登身上。
          <br>
           <br/>
           2015年5月30日博拜登去世，亨特拜登乘虚而入就跟嫂子哈莉滚了床单，彼时博拜登仅仅去世才两天，亨特没等他哥过完“头七”就对嫂子下手，亨特与嫂子发生关系时还没有与妻子凯瑟琳离婚，两人只是关系不和睦。
           <br/>
           <br/>
           注意时间线，2015年7月，亨特与凯瑟琳分居，在亨特“硬盘门”中，凯瑟琳在邮件中抱怨道：“博叔叔死的第二天，你爸爸就睡了哈莉婶婶”，这是原话。拜登家族中上到拜登下到拜登的孙子们都知道了此事，妥妥的家庭丑闻。
           <br/>
           <br/>
           但是伦理丑闻经亨特之口被说的那叫一个清新脱俗，亨特称被嫂子哈莉大眼睛和闪耀的猫式的微笑所迷惑了。亨特认为嫂子对他有“催眠性和令人难以置信的诱惑力”。从那时候开始，亨特就开始在大嫂家里过夜，不管自己的妻子和孩子，而是一心一意的去照顾大哥"托付"给他的大嫂和自己的侄子侄女。
           <br/>
           <br/>
           好景不长，痛失旧爱又得新欢的哈莉没过几天安稳日子，也就一年后，亨特就完全不顾大哥的“托付”又移情别恋了，且下手的不是别人，正是嫂子哈莉的姐姐和妹妹，亨特也算个撩妹高手，他不断的给嫂子哈莉的姐妹两人发送色情短信，据说还险些攻陷了哈莉姐姐伊丽莎白的心理防线。话说哈莉头也够铁，自己搭进去不说还险些送自己的两个姐妹进虎口。
           <br/>
           <br/>
           随后，拜登深知藏的再深也有被挖出的风险，所幸自曝家丑并祝福亨特与哈莉幸福快乐每一天。但事与愿违，2017年底，按耐不住寂寞的亨特再度出手成功拿下一名风尘女子，该女子是一位脱衣舞女，脱衣舞女也是过来人，她成功怀上了亨特的孩子最后生下一子，起初亨特死活不认，但是迫于形势还得乖乖认下孩子，不过在给孩子取名字的过程中，亨特再次展现了他的脑回路，他给私生子取名为博，意思是要纪念去世的哥哥，真是太同情博拜登了。
           <br/>
           <br/>
           2018年7月，嫂子哈莉的妹妹还是没有逃过亨特的魔掌，两人还住到了一起。同时，亨特还给嫂子哈莉的姐姐也租了豪华公寓，最终嫂子哈莉的两位姐妹都没逃过一劫。2019年4月，心灰意冷的哈莉与亨特分手，一个月后亨特与南非模特梅丽莎闪婚。
           <br/>
           <br/>
           就在如此纷乱复杂的关系下，疑似薄熙来之女的薄甜甜入局了，“硬盘门”中显示鲍佳琪与亨特的来访邮件中尽显暧昧，鲍佳琪简直就是亨特的小迷妹，对亨特崇拜的不行。不过，生性浪荡的亨特本就风流，不知道鲍佳琪左右了亨特多少？
           <br/>
           <br/>
           亨特的新书中，勾引大嫂的事他认了，以往的风流债他也认了，吸毒成瘾还制造毒品的事情他也认了，酗酒、借别人尿液用于药检想进美军海军的事依旧也是认了，但是借着他爹拜登的权势大搞黑金政治的事情，包括乌克兰事件亨特至今不认，即便媒体又开始挖亨特的“硬盘门”，亨特依旧我自岿然不动。
           <br/>
           <br/>
           总之，涉嫌犯罪的事亨特一概不认，凡涉及“私德”的事情他一笔带过，那么亨特到底有没有犯罪，大家自己品品，仔细品品。
