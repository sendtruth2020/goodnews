 <p>
  【看中国2021年2月6日讯】（看中国记者真瑜采访报道）美国时间2月4下午，
  <strong>
   拜登
  </strong>
  发表了上任以来的首次
  <strong>
   外交政策
  </strong>
  谈话，除了对外宣告“美国回来了”，最引人注目的是
  <strong>
   拜登政府
  </strong>
  的对话政策。拜登称，美国将直面
  <strong>
   中共
  </strong>
  滥用经济地位，反制其在人权、知识产权和全球治理方面的好斗和高压行为。但是，当与
  <strong>
   美国
  </strong>
  人利益一致时，愿意与中方合作。拜登此话引起多方猜测，美中关系到底何去何从。
  <br>
   <br>
    拜登此次讲话把中方形容为竞争者，替代了
    <span href="https://www.secretchina.com/news/gb/tag/川普" target="_blank">
     川普
    </span>
    (特朗普)时代的中共是美国“头号敌人”的观点。拜登的讲话中，外媒过分的注重了前半段而刻意忽略了拜登迫切的想与中共修好的愿望。其中，拜登外交政策要重塑美国在盟友中的领导地位，他将与盟友们一起遏制中共，然而法国立刻回应不买拜登的账，马克龙表示，虽然欧盟与美国的价值观相近，但不会和美国一道对付中共。相比与拜登的外交政策，川普就显得极为简单，就是美国先挽起袖子干，给盟友们做表率，盟友们看到美国大哥都玩真的了，自然而然也会跟上。
    <span id="hideid" name="hideid" style="color:red;display:none;">
     <span href="https://www.secretchina.com">
     </span>
    </span>
   </br>
  </br>
 </p>
 <p>
  美国南卡罗莱纳大学艾肯商学院讲座教授谢田博士认为，川普时期美国称中共是战略竞争对手，
  <span href="https://www.secretchina.com/news/gb/tag/拜登" target="_blank">
   拜登
  </span>
  这次称中共是严峻的竞争对手，这是错误的。拜登把中共看作是竞争对手而不是敌人的时候，就意味着要跟中共合作，实际上是把川普政府千千辛万苦搭建的灭共围墙拆开一个口子，之前川普对付中共的战略性努力，现在很可能会被拜登毁掉。
  <br>
   <br>
    谢田博士表示，老派政客出身的拜登没有真正认识到中共政权的邪恶程度，他基本上要回到奥巴马时期对中共的绥靖策略。从执行步骤上看，拜登启用了大部分奥巴马时期的官员，可以遇见的是拜登对中共经济、政治、外交、科技、人权等问题的攻势肯定会有所减缓，不排除要走回头路的可能。另外，民主党掌控了白宫和参众两院，拜登在外交上将更有话语权，没有什么顾忌，如果拜登真心想放中共一马，他是有可操作空间的。
    <br>
     <br>
      谢田博士指出，拜登想联合欧洲的盟友一起对付中共，本质上是树立美国对欧盟各国的领袖作用。实际上，欧盟国家也有联合对付中共的行为，但是欧洲国家的动作非常慢，中共又善于利用各种手段将反共集团各个击破，所以欧洲国家对中共的态度总是左右摇摆，要真正打到中共七寸，还需要率先从美国开始做起。
      <br>
       <br>
        另外，外媒评价拜登的外交政策是大国回归，叫板中俄。外媒认为拜登将强硬应对中俄的挑战，美国会一如既往的遏制专制主义。与外媒认为美国将强硬对华的乐观不同，中共央视新闻将拜登的讲话掐头去尾，只强调拜登已经准备好要与中国加强合作。中共外交部的回应极度谄媚，发言人汪文斌说，中方致力于同美方发展不冲突不对抗、相互尊重、合作共赢的关系。期盼美国采取积极建设性的对华政策，同中方相向而行，聚焦合作，管控分歧，推动中美关系健康稳定向前发展。
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  对此谢田博士表示，世界媒体认可拜登的对华外交政策是基于他们的观点和利益。中共媒体通过各种管道放出声音，实际上都是在试探拜登的底线。拜登一直是中共希望获胜的候选人，中共让川普下台的愿望已经达成，他们现在的目标是如何软硬兼施迫使美国放其一马。从拜登的外交政策讲话看，其实透露了他的对华政策存在非常大的不确定性，拜登也有顾虑，他不想得罪选民，又想从中共那里获得好处，所以他的发言才会显得份量不足。
  <br/>
  <br/>
  还有一点值得关注，拜登时政纲领关注的是气候、同性恋平权、种族歧视上，尤其气候问题是拜登团队最关注的，如果要在气候问题上有所作为，拜登肯定会回到巴黎气候协议，这就难免要跟中共合作，中共也必然会以此作为筹码要求拜登放宽贸易、科技等方面的限制。拜登的施政策略本身有硬伤，内政影响到外交恐怕在所难免。
  <br/>
  <br/>
  拜登发表对华政策讲话前不久，在官方话语体系中删除了“中国病毒”、“
  <span href="https://www.secretchina.com/news/gb/tag/中共病毒" target="_blank">
   中共病毒
  </span>
  (又称新冠状病毒，COVID-19)”等词汇。众所周知，病毒来自中国，因为中共的刻意隐瞒造成病毒席卷全球，到目前为止有2百多万人失去生命，以美国为首的西方国家纷纷出来要向中共索赔，现在拜登能改变对病毒的称谓，他日放松甚至放弃索赔也未必不可能。
  <br/>
  <br/>
  与其说拜登的外交政策开启了美国外交政策的新时代，不如说拜登让美国的外交一夜回到绥靖时期。说到底，美国正站在历史的十字路口，拜登如何择选针对中共的政策，关系到世界民众如何再次定义民主灯塔问题，所以要谨慎选正路。
