<p class="p1">
 当天的暴风雪，是继
 <span class="s1">
  2016
 </span>
 年
 <span class="s1">
  2
 </span>
 月以来纽约市最大的一场，纽约中央公园录得
 <span class="s1">
  16.8
 </span>
 英寸的降雪，阵风时速达
 <span class="s1">
  50
 </span>
 英里
 <span class="s1">
  /
 </span>
 小时。暴雪在狂风中，像刀子一样打在人的脸上，吹得人眼难以睁开，出行变得非常困难。
</p>
<p class="p1">
 在这样恶劣的天气中，一首“法轮大法好、真善忍好”的《九字真言歌》却在风雪中回荡：
</p>
<p class="p1">
 亲爱的朋友啊，你可知道，
 <span class="s1">
  <br/>
 </span>
 我们的生命来自神的国度，
 <span class="s1">
  <br/>
 </span>
 创世主慈悲末世救度，
 <span class="s1">
  <br/>
 </span>
 默念九字真言，难中保平安，
 <span class="s1">
  <br/>
 </span>
 法轮大法好，真善忍好。
</p>
<p class="p1">
 亲爱的朋友啊，请听我说，
 <span class="s1">
  <br/>
 </span>
 法轮大法是正法，佛光普照，
 <span class="s1">
  <br/>
 </span>
 善念一出，天地皆知，
 <span class="s1">
  <br/>
 </span>
 诚念九字真言，难中保平安，
 <span class="s1">
  <br/>
 </span>
 法轮大法好，真善忍好。
</p>
<p class="p1">
 法轮功学员向广大民众传递真相的心愿，并没有因为暴风雪而消退；铲积雪、架棚子，真相点照常在早上
 <span class="s1">
  10
 </span>
 点开放。过往路人再次看到这道熟悉的风景线，听到熟悉的旋律和歌声。
</p>
<p class="p1">
 这是一个顶住了中共
 <span class="s1">
  20
 </span>
 年抹黑打压，坚忍不屈反迫害的群体，十多年来，无论狂风骤雨还是冰天雪地，纽约法拉盛的法轮功真相点从未间断。
</p>
<p class="p1">
 在法拉盛图书馆前的真相点上，早上
 <span class="s1">
  11
 </span>
 点多，风雪最大时，一位行人路过，并向真相点的义工索要真相资料，接过法轮功学员手制、写有法轮功大法好的小莲花后，对义工表示感谢；一名西人，站在真相点旁，静静地聆听《九字真言歌》，他说歌曲“真好听”，他接过了法轮功的传单和“法轮大法好”的小莲花，并对法轮功学员表示感谢。
</p>
<h4 class="p1">
 “这么大的雪，停一天不行吗？”
</h4>
<p class="p1">
 法轮功学员陈丹表示，虽然雪这么大，但常人都有出门上班的，我们为什么不坚持呢？“我就想，如果今天是打工挣钱，你去不去？你不还是得去吗？何况我们是修炼人，是在救人呢？”所以虽然当天本不是陈丹当班，但当收到信息问谁可以去真相点时，她毫不犹豫地就来了。
</p>
<p class="p1">
 “这么大的雪，停一天不行吗？”记者问。法轮功学员郭小芳回答，“我觉得有机会我就很珍惜。作为大法弟子，现在瘟疫来了，人类的劫难来了，很多人处于危险之中，大法弟子就是要抓紧时间救人，告诉他们记住法轮大法好，能救一个是一个。”
</p>
<p class="p2">
 <span class="s2">
  不畏严寒，郭小芳在风雪中伫立。有过路的行人对她说：“
 </span>
 You are very nice, You are very beautiful
 <span class="s2">
  ”。
 </span>
</p>
<h4 class="p1">
 “我好了，也希望大家都好”
</h4>
<p class="p1">
 <span class="s1">
  66
 </span>
 岁的法轮功学员胡英明正在铲雪，她感觉身体比在年轻时还要好。她还在打工，做护理员，干活麻利，力气大，别人都说看不出她有这么大年纪。
</p>
<p class="p1">
 胡英明说：“我
 <span class="s1">
  2003
 </span>
 年开始修炼，刚学功第一天，原来严重的胃病就再也没有发作过。我要把法轮大法的美好与大家分享。”
</p>
<p class="p1">
 在法拉盛地铁口真相点上，张丽娟说：
 <span class="s1">
  “
 </span>
 修炼后身心受益巨大，所以也希望大家都从法轮大法中受益。
 <span class="s1">
  ”
 </span>
</p>
<p class="p1">
 张丽娟原来患有严重的乳腺小叶增生。
 <span class="s1">
  2003
 </span>
 年，中共迫害法轮功最高峰时，她的同事见她每天捧着药罐子，问她敢不敢学法轮功？“‘共产党不让炼，你敢不敢炼？’她说：‘只要身体好，有什么不敢炼的？！’”
</p>
<p class="p1">
 炼功两个月后，张丽娟的医生见她没去看病了，给她打电话，“她以为我死了。我去她那检查，她一摸我的乳房，非常柔软，大吃一惊，问：你吃了什么药，你的小叶增生怎么全部没有了！”
</p>
<p class="p1">
 “我原来的乳腺增生非常严重，一块块硬硬的，我跟她说，我没吃什么药，就是炼了法轮功。她就指着其他病人说，你跟他们说说，你炼法轮功，乳腺增生没有了！”
</p>
<p class="p1">
 “我好了，我也希望大家好。”张丽娟从
 <span class="s1">
  2015
 </span>
 年开始在真相点讲法轮功真相，从没停过。
</p>
<p class="p1">
 法轮功学员刘锡铜说：“疫情下，现在是救人最关键的时候，抓紧每一分钟，抓住每一个机会。”
</p>
