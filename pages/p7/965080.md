 <p>
  一个人的一生中, 某个或某几个关键时刻, 由于一念之差, 错过了机会, 或做错了一件事, 后悔莫及。假如, 这仅仅是一个人或一个家庭的事, 造成损失或伤害的那是一个人或一个家庭的. 但, 一个人或一些人做的或做过的事情, 关系到一个民族的未来和一个国家的长治久安, 总该值得我们深思吧.
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  <br>
   那一天, 你站在高高的城楼上
   <br>
    说, 我们从此站起来了, 可我哪里敢啊
    <br>
     你是站起来了, 我只能趴着
     <br>
      本来, 民主的会议协商
      <br>
       定下了民主的宪政大章
       <br>
        但你不屑一顾, 弃之一旁
        <br/>
        那个要命的关键时刻啊
        <br/>
        智者的理论怎敌得过小人的捧场
        <br/>
        再说, 你手中有的是枪, 还有
        <br/>
        几个死了的洋大胡子作了你的排场
        <br/>
        你的血液里呀, 帝王的细胞在增长
        <br/>
        虽说不称帝, 换件新衣, 可你骨子里呀
        <br/>
        要的是孤家寡人, 权力至高无上
        <br/>
        几千年的宫药啊, 仅仅换了汤
        <br/>
        自由民主的幌子打够了
        <br/>
        你把它扔回了大西洋
        <br/>
        连同你恰同学少年的理想
        <br/>
        这一念之差哟, 千千万万的鲜血白白流淌
        <br/>
        你想创造经济的奇迹来证明一个幻想
        <br/>
        炼铁链钢, 还有那三面红旗飘扬
        <br/>
        善良的百姓哪里知道啊
        <br/>
        他们的血汗, 流进了狂人的痴想
        <br/>
        人造的灾难, 有多少生灵去西天游荡
        <br/>
        革命, 口号是那么的响亮
        <br/>
        革命, 你还没有过瘾
        <br/>
        无数无知的热血被你瞬间膨胀
        <br/>
        整垮整死了多少帮你实现帝梦的老将
        <br/>
        几千年沉淀的优秀的文化文明啊
        <br/>
        几乎就毁在了你的手上
        <br/>
        毛头青年用过了, 你指示他们
        <br/>
        去接受再教育, 上山下乡
        <br/>
        你该安息啦, 因为你的躯壳
        <br/>
        今天还有人排队观赏
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  那一天, 你知道有多少人在翘望
  <br/>
  拨乱反正, 可什么是正
  <br/>
  回归忠良, 可什么是忠什么是良
  <br/>
  被长期愚弄的百姓啊
  <br/>
  好像看到了又一颗救星又一颗太阳
  <br/>
  本该, 重用被弃的自由民主的宪政大法
  <br/>
  本该, 重建纲常用来自大西洋的思想
  <br/>
  可你要四个坚持, 维护功成名就的故王
  <br/>
  和那谁也搞不清楚的所谓的思想
  <br/>
  这一念之差哟, 你的被整死的战友们
  <br/>
  在西天气得直跺脚, 大喊大叫死得窝囊
  <br/>
  那些为你抗争于广场的热血心肠哟
  <br/>
  被你灌满了冰冷的陈汤
  <br/>
  你知道发展是硬道理, 知识就是力量
  <br/>
  怎么让人觉得, 那是为了太后的红色衣裳
  <br/>
  为什么不敢要第五个现代化
  <br/>
  难道就你一人文明
  <br/>
  几千年文明造就的百姓全都是猪羊
  <br/>
  广场的毛头青年用过了, 你指示他们
  <br/>
  快快去蹲一蹲红色的牢房
  <br/>
  经济好些了, 上层建筑却垮了
  <br/>
  你不让一个再一个你党的总书记改造破碎的瓦房
  <br/>
  洋大胡子的基本原理都不顾了
  <br/>
  还谈什么主义, 什么思想
  <br/>
  这难道不是天下最大的荒唐
  <br/>
  多少年轻的生命哟, 倒在那血腥的广场
  <br/>
  难道, 难道, 这就是你青年时的理想
  <br/>
  你该安息啦, 因为你摸过的那块石头
  <br/>
  今天还有人在顶礼膜拜, 在歌唱
 </p>
 <p>
  那一天, 你知道有多少人泪淌
  <br/>
  你来到广场, 你说你来晚了
  <br/>
  可孩子们哪里知道你慈爱而无奈的心肠
  <br/>
  那一双双童真的眼, 紧紧地盯着你
  <br/>
  难道你真的懊悔, 真的绝望
  <br/>
  难道你那只百姓吃粮要找你的手臂挥不起来
  <br/>
  难道你害怕坐你自己的党为你备下的班房
  <br/>
  难道你不知道 Nelson Mandela 可以是你心灵的伙伴
  <br/>
  难道你不知道独裁制度或迟或早的下场
  <br/>
  这一念之差哟, 你让小人得了志更加的猖狂
  <br/>
  自由民主抗争的勇士们被关进了牢房
  <br/>
  多少本该为国贡献的精英, 在异国他乡流亡
  <br/>
  本来, 你可以写下历史, 以总书记的身份
  <br/>
  把你党给你的牢底来坐穿
  <br/>
  本来, 你可以成为自由民主的先驱
  <br/>
  你可以永远是中华的好儿郎
  <br/>
  你可以是百姓心中永远不落的紫色太阳
  <br/>
  天堂的你, 怎能瞑目啊
  <br/>
  大大小小的贪官污吏, 久割又长
  <br/>
  无数的百姓哟, 病无所医, 老无所养
  <br/>
  人灾不断, 多少无辜的生命遭殃
  <br/>
  哺育万物的河川哟, 污水在流淌
  <br/>
  这一切的一切
  <br/>
  难道你能安详在天堂
 </p>
 <p>
  啊, 拷问灵魂的鞭子哟 (*)
  <br/>
  重重地抽吧, 噼哩啪啦地响
  <br/>
  抽碎那宫廷的毒药瓶
  <br/>
  抽来民主自由的雨露阳光
  <br/>
  抽去那独裁者的谎言
  <br/>
  抽出人之初的童真, 追求真理的信仰
  <br/>
  抽去邪念, 恶念, 帝王之念, 一党之念
  <br/>
  抽来正义, 慈善, 民主理念, 民富国强
  <br/>
  啊, 拷问灵魂的鞭子哟
  <br/>
  你的节奏多么激动人心
  <br/>
  你的旋律多么荡气回肠
  <br/>
  我祈祷
  <br/>
  让自由的歌声吹到大地的每一个角落
  <br/>
  让民主的旗帜在每一个人的心中飘扬
  <br/>
  <br/>
  <br/>
  <br/>
  (*) 诗人尚飞鹏的"寻找拷问灵魂的鞭子"
  <br/>
  https://www.poemhunter.com/poem/search-for-the-whip-of-soul-torturing/
