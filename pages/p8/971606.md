 <p>
  家中的孩童喜欢吃
  <span href="https://www.secretchina.com/news/gb/tag/糖果" target="_blank">
   糖果
  </span>
  吗？市售的糖果原料几乎全是化学品，也没有营养成分，不如一起来学手工糖果的制作方法，除了可以全家同乐，也能兼顾健康与美味。
 </p>
 <p>
  <strong>
   芝麻软糖
  </strong>
 </p>
 <p>
  原料：
  <br>
   <br>
    细砂糖120克、橄榄油20克、熟白芝麻和熟黑芝麻各120克、洋菜粉8克、水麦芽60克、水60克、盐5克、太白粉。
    <br>
     <br>
      制做方法：
      <br>
       <br>
        1.将洋菜粉先放入水中浸泡，静置半小时后，放入锅里以大火快煮2分钟，随即关火。
       </br>
      </br>
     </br>
    </br>
   </br>
  </br>
 </p>
 <p>
  2.放水麦芽60克、120克细砂糖和5克盐入锅，这时请记得要一边搅拌一边煮，直到锅内水麦芽温度到110度就可以了。
  <br>
   <br>
    3.另外用一锅具，将将水和太白粉搅拌均匀，均匀后，再分多次加入原先锅内的水麦芽，这时同样必须搅拌，煮至120度时方可关火。
    <br/>
    <br/>
    4.此时加入橄榄油，一同拌匀。
    <br/>
    <br/>
    5.倒入黑芝麻、白芝麻一同拌匀。
    <br/>
    <br/>
    6.在干净的桌面或是料理台上，铺上保鲜膜或锡箔纸，纸上刷上一点油。
    <br/>
    <br/>
    7.用软刮板摊开并压平将整个。
    <br/>
    <br/>
    8.等糖体冷却之后，依照个人喜好切成小条状或是方块状，最后再进行包装。
    <br/>
    <br/>
    9.当然，可放入不同坚果做成不同口感的白芝麻糖、花生糖、杏仁糖等。
   </br>
  </br>
 </p>

  <strong>
   ​水果口味法式软糖
  </strong>
 </p>

 <p>
  原料：柳橙2颗、葡萄20颗、柠檬汁5毫升、鱼胶粉20克、麦芽糖120克、白糖40克、水40克。
  <br/>
  <br/>
  制做方法：
  <br/>
  <br/>
  1.将柳橙纵向切对半，挤出柳橙汁，接着用筛子过滤掉果粒和籽。
  <br/>
  <br/>
  2.因为要要将果皮一起打成汁，糖果颜色才会好看，尽量挑选有机、无毒、无农药的葡萄。洗净葡萄外皮后，放入果汁机打成果汁。
  <br/>
  <br/>
  3.鱼胶粉分别放入葡萄汁和柳橙汁中，静置5至10分钟，使鱼胶粉变软。
  <br/>
  <br/>
  4.以小火煮水，再加入麦芽糖及白糖，这时要不断搅拌，温度达到115度即可关火。
  <br/>
  <br/>
  5.另起一锅，将鱼胶粉隔水加热，使鱼胶粉完全融化。
  <br/>
  <br/>
  6.两种口味的鱼胶粉已融化，这时掺入柠檬汁，再与步骤4中煮好的糖浆充分搅拌均匀。
  <br/>
  <br/>
  7..将两种口味的混合物倒入模具，软糖冷却会凝固，但不够完全，必须放入冰箱冷藏。
  <br/>
  <br/>
  8.凝固加冷藏的时间可能稍微长一些，不时可以用竹签刺看看，或是尝一下口感，差不多30小时可以完全凝固。
 </p>

  <strong>
   果汁硬脆糖
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  原料：白糖300克，新鲜草莓500克，水200克。
  <br/>
  <br/>
  制做方法：
  <br/>
  <br/>
  1.用活水将草莓洗净，同样的，因为草莓很可能含有较多的农药，所以尽量挑选无毒、无农药的有机草莓来制做
  <span href="https://www.secretchina.com/news/gb/tag/天然" target="_blank">
   天然
  </span>
  糖果为好。
  <br/>
  <br/>
  2.将洗清洗过后的草莓，放入果汁机中，榨成草莓果汁。
  <br/>
  <br/>
  3.使用筛子过滤刚刚榨出来的草莓汁，让果肉渣与果汁分离。
  <br/>
  <br/>
  4.煮滚水后倒入白糖，用红糖也可以。
  <br/>
  <br/>
  5.待糖彻底融化后，即成糖浆，这时可以倒入草莓汁。
  <br/>
  <br/>
  6.以小火煮草莓糖浆，等糖浆成有泡沫的粘稠状便关火。
  <br/>
  <br/>
  7.倒入各种模具后再行冷藏。
  <br/>
  <br/>
  8.草莓可以替换成你想要的任何口味的水果，但因为完全没有添加香料，只靠水果香气，如果水果的量不足，香气也会略淡，所以请选择香气浓烈的水果，并使用足够的量来榨汁。
