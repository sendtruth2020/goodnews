 <p>
  在许多人的传统观念里，把吃不完的饭菜放
  <span href="https://www.secretchina.com/news/gb/tag/冰箱" target="_blank">
   冰箱
  </span>
  保存，下一餐再热来吃，可说是日常基本操作。只不过“热食能不能放冰箱？”就有很大的争议。

 <p>
  在许多长辈的观念中，很多都有着“饭菜应该等凉了再放冰箱”这样的想法，理由是——热食放冰箱容易坏，也容易伤害到冰箱。
 </p>
 <p>
  但事实真的是如此吗？要解开这个谜，我们得先明白3个问题：
 </p>
 <p>
  1、等待放凉的食物，会发生什么？
 </p>
 <p>
  2、趁热放冰箱，真的会伤冰箱吗？
 </p>
 <p>
  3、怎么放合适？
 </p>

  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  通常食物上都会沾有“
  <strong>
   大肠杆菌
  </strong>
  ”这种常见的细菌。它们平时对人无害，但若达到一定数量，就会转变成让人生病的“条件致病菌”。当然也有
  <span href="https://zh.wikipedia.org/wiki/%E9%87%91%E9%BB%83%E8%89%B2%E8%91%A1%E8%90%84%E7%90%83%E8%8F%8C">
   金黄色葡萄球菌
  </span>
  、
  <span href="https://zh.wikipedia.org/wiki/%E6%B2%99%E9%96%80%E6%B0%8F%E8%8F%8C%E5%B1%AC">
   沙门氏菌
  </span>
  等，它们在数量很少时，就已经是很厉害的“强毒力致病菌”
 </p>

  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  这些
  <span href="https://www.secretchina.com/news/gb/tag/细菌" target="_blank">
   细菌
  </span>
  若大量增殖，吃进肚子就会出问题，可能导致细胞功能紊乱、毒血症，引发腹痛、腹泻、呕吐等消化道症状。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  影响细菌增殖的因素有很多，像食物的营养成分、酸碱度、含水量、烹饪的方法都有关系，但影响最大也最普遍的还是在于“温度”。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  研究证明，大多数细菌最爱的温度是位于4℃～60℃，因此这个区间也被称为食物的“危险温度区”。一旦超过60℃，绝大多数细菌无法存活；若低于4℃，细菌还是会增殖，只是速度就明显趋于缓慢。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  巧合的是，人的体内温度（内脏温度）约为35℃～37℃，这个范围正好是沙门氏菌、金黄色葡萄球菌等常见致病菌，最适合的生长温度。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  <strong>
   ●细菌在不同温度的增殖速度不同，4℃以下则无明显增长
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

  所以了解这个原理后，想降低细菌的繁殖速度，就应该要让食物尽快降到4℃以下保存。因为多在室温下放一分钟，反而多一分危险。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  而食物想放冰箱，就要确实做到2点：“尽快”加“密封”。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  那么“尽快”到底要多快？
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  简单来说，就算是一盘刚出炉的热菜，只要不准备吃，就可以马上装进密封盒子里，放进冰箱。为何采取密封盒保存？那是因为可以让食物不沾染其它味道，还能减少水分蒸发。此外，除了刚做好的菜，买回来没有要马上吃的菜，也要尽快放冰箱。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  美国食品药品监督管理局就建议大家不要超过2小时。像生肉、海鲜、蛋类及其他必须冷藏的食品，都别在室温下放置超过2小时；若室温超过32℃，更不能放超过1小时。
 </p>

  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  <strong>
   ●热菜直接放进去，伤冰箱吗？
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">
   </div>
  </div>
  <div>
   <center>
    <div id="div-gpt-ad-1589559869784-0">
    </div>
   </center>
  </div>
 </center>
 <p>
  看完上面的说明，相信这应该是许多人关心的问题，毕竟很多长辈就是这么教我们的。但事实是影响不大，因为都在冰箱的正常运行范围内。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  当需要带走的热量越多，冰箱的“负担”自然就越大，消耗的电力也就越多。另外如果热菜没有密封好，那么就会有大量水汽蒸发，凝结在冰箱的冷凝管上，这也会影响冰箱的运作。而这，也就是许多人认为“不能把热菜直接放冰箱”的理由。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">
   </div>
  </div>
  <div>
   <center>
    <div id="div-gpt-ad-1589559869784-0">

   </center>
  </div>
 </center>
 <p>
  所以热菜放入冰箱时，的确会短暂提高冰箱温度。但冰箱也有装设温度传感器，一察觉温度升高，就会启动降温程序，尽快让冷藏室恢复低温状态。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">
   </div>
  </div>
  <div>
   <center>
    <div id="div-gpt-ad-1589559869784-0">
    </div>
   </center>
  </div>
 </center>
 <p>
  因此除了关心对冰箱“有无影响”，我们反而更该关心带来“多少影响”。毕竟冰箱的“负担”不仅取决于食物温度，更取决于需要带走的热量。热菜“直接放”，难免会增加冰箱负担，但也仅是多耗一点点电，都在冰箱的正常运作范围内。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  <strong>
   ●热菜温度高，会不会让旁边的菜坏掉？
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  可能也有人会联想到这个问题，但这事同样不必太担心。因为在这段降温的时间里，确实会让周围的食物温度“略微”上升，但影响很小，也就等同开了几次冰箱门而已。这种临时升高的温度，并不会让冷藏室的温度发生质的变化。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  总之，冰箱比人们想象中的运作还要稳定。所以为了家人健康着想，热的饭菜是可以直接放入冰箱的。只是要注意的是，为了避免冰箱发生串味和结霜的现象，最好要用保鲜膜密封，或者放进保鲜盒里分开存放。
