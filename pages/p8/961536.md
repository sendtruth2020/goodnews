 <p>
  中国新年即将来临，许多人在大扫除的同时，也会对爱车进行清理。那么该如何掌握好
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/汽车" target="_blank">
    汽车
   </span>
  </strong>
  的
  <strong>
   清洗
  </strong>
  技巧，让爱车保持光亮如新呢？

 <p>
  <strong>
   1
  </strong>
  <strong>
   、尽快清除会腐蚀
   <span href="https://www.secretchina.com/news/gb/tag/烤漆" target="_blank">
    烤漆
   </span>
   的东西
  </strong>
 </p>
 <p>
  许多车子在室外停久了，对漆面也会造成损伤。例如把车停在树下，看似能略微遮阳。但如果树上有蚜虫的话，因为蚜虫会从树叶中吸取汁液并分泌带黏性的糖液，当这些排泄物滴到车上后，被阳光照久以后，也会损伤漆面。
 </p>
 <p>
  除了蚜虫等昆虫外，
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/鸟粪" target="_blank">
    鸟粪
   </span>
  </strong>
  也具有腐蚀性。而这些腐蚀性物质在漆面停留的时间越长，造成的损害就越大。而这种损坏是不可逆的，只能找专业维修。
 </p>
 <p>
  因此如果有昆虫或动物的排泄物黏在汽车漆面上，请尽快处理掉。最好可以使用专门去除昆虫及其残留物的清洁剂，这样不但能够有效防止车漆遭到损害，洗车时也容易擦去黏附物。
 </p>
 <p>
  <strong>
   2
  </strong>
  <strong>
   、平时忽视保养 后续维护费用更高
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  车漆是很多车主忽视的一个保养项目，也不会定期护理车身漆面。根据经验，想保护车漆，至少每周要洗一次车。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

  <span href="https://www.secretchina.com/news/gb/tag/清洗" target="_blank">
   清洗
  </span>

  <strong>
   3
  </strong>
  <strong>
   、洗车站的工具是否清洁？
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  在加油站能看见不少用刷子或布条来洗车的机器，但是要注意这些刷子或抹布是否干净完整。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  假如尼龙线过于脆弱易撕裂、或是布条很脏，那么灰尘、沙粒可能会藏在其中。这样洗车时，这些清洁工具就有可能像砂纸一样损坏漆面。所以平时到加油站洗车时，要仔细观察那些工具，留心它们本身的清洁和保养状况。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  至于清洗后是否有必要上蜡？这绝对有必要，因为蜡能封住车子的烤漆，保护车漆免受外部影响。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  那么上蜡保养该多久做一次？只要雨水能从车身正常滑落，就不必进行再次打蜡。若汽车常停在车库中，则蜡的停留时间会比在车棚或露天场所更长。若拿捏不定的话，不妨每年打2次蜡，时间可以定在春季跟冬天到来前。
 </p>

  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

  <strong>
   、自己洗车要注意什么？
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 </center>
 <p>
  喜欢自己洗车是很好的习惯，也是对爱车最好的一种体贴照顾方式了。但要注意使用pH值低的温和清洁剂，还有柔软、干净的海绵、刷子和布。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  在用高压水枪时，若水压过高，也会损坏漆面。同时每款清洗设备有所差异，加上车款不同，所以喷头与汽车漆面之间的正确距离也不同。根据经验，若水枪的喷水会让手觉得疼，那么它也会对汽车造成伤害。
 </p>

  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  <strong>
   5
  </strong>
  <strong>
   、抛光车身
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  觉得自己的车子不像新买的有光泽吗？甚至可以看到轻微的划痕？若是这样，那就应该给车子抛光了。汽车抛光剂含超细磨料，可轻软地清洁车漆表面，却不会造成损坏。抛光剂还能持续保护漆面，减少雨水、雪对车漆的影响。
 </p>

  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

   、修补漆面

  </strong>
  <strong>
   、哑光漆的特殊保养方式
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  如果车身表面是哑光漆，那就不能采用抛光，因为抛光剂中的研磨剂会去除哑光漆独特的粗糙表面。此外，频繁使用洗车刷、蜡密封剂也会破坏哑光漆。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  因此哑光漆需要采用完全不同的方式来护理，最好用手或高压水枪，并且使用专门的清洁和保养产品。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
