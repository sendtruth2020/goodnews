 <p>
  2021年，网友总结的新6大关系，精辟而又准确，不妨来看看。
  <br>
   <br>
    <strong>
     一、离开就断了的，是
     <span href="https://www.secretchina.com/news/gb/tag/工作" target="_blank">
      工作
     </span>
     关系
    </strong>
    <br>
     <br>
      古人有句非常经典的话：“以利相交，利尽则散；以势相交，势去则倾；以权相交，权失则弃；以义相交，地久天长。”
      <br>
       <br>
        打开手机通讯录，里面有上千个联系人，在这成百上千人之中，70%是曾经工作上的伙伴，除了工作内容之外，似乎再也找不到可以交流的话题。
        <br>
         <br>
          而工作上的伙伴，大多数是利益相互者，一旦离开了利益圈，我们似乎没有理由去来往。
          <br/>
          <br/>
          工作关系，离了就断，断了就再也不见。
          <br/>
          <br/>
          <strong>
           二、距离也断不了的，是亲属关系
          </strong>
          <br/>
          <br/>
          或许时间和距离可以阻隔人与人之间的情感，但惟一无法割断的便是血缘之亲。
          <br/>
          <br/>
          亲情不会因为靠拢就愈加浓烈，也不会因为距离远而逐渐疏离。
          <br/>
          <br/>
          <strong>
           三、有事才想起的，是利用关系
          </strong>
          <br/>
          <br/>
          中国有句古话：门前放根讨饭棍，亲戚故友不上门。
          <br/>
          <br/>
          有些人，你风光时，他希望你能帮衬自己；你落魄了，他却比陌生人都还会冷言冷语。
          <br/>
          <br/>
          与人相处，最忌讳的就是利用。
          <br/>
          <br/>
          <strong>
           四、没事约吃饭的，是朋友关系
          </strong>
          <br/>
          <br/>
          不是你的亲人，没有血缘，但是彼此碰撞出了精神上的默契。因为有他的存在，让你在这世上，不再寂寞，不再孤独。
          <br/>
          <br/>
          是的，朋友就是有事没事，随时可以联系的人。
          <br/>
          <br/>
          要记得那些大雨中为你撑伞的人，帮你挡住外来之物的人，黑暗中默默抱紧你的人，逗你笑的人，陪你彻夜聊天的人，坐车来看望你的人，说想念你的人。
          <br/>
          <br/>
          是这些人组成你生命中一点一滴的温暖，是这些温暖使你远离阴霾。
          <br/>
          <br/>
          <strong>
           五、付出不求回报的，是母子关系
          </strong>
          <br/>
          <br/>
          一天晚上，一位妈妈正在准备晚饭。
          <br/>
          <br/>
          10岁的儿子走进厨房递给她一张纸，上面写着：
          <br/>
          <br/>
          割草，5元；
          <br/>
          <br/>
          这一周整理自己的床铺，1元；
          <br/>
          <br/>
          去商店，5元；
          <br/>
          <br/>
          你去购物，我照看小弟弟，2.5元；
          <br/>
          <br/>
          倒垃圾，1元；
          <br/>
          <br/>
          取得了优秀的成绩单，5元；
          <br/>
          <br/>
          还有打扫院子，2元。
          <br/>
          <br/>
          看着孩子满怀期待地站在那里，千万个记忆一瞬间闪过妈妈的脑海。
          <br/>
          <br/>
          妈妈接过那张纸，翻到背面，在上面写道：
          <br/>
          <br/>
          怀你10个月，免费；
          <br/>
          <br/>
          为你熬夜，请医生为你看病，免费；
          <br/>
          <br/>
          多年来花在你身上的时光、为了你流过的泪、抚养你成长所付出的一切，免费；
          <br/>
          <br/>
          日夜为你担忧，将来还要为你操心，免费；
          <br/>
          <br/>
          给你忠告和教你知识，供你上学，免费；
          <br/>
          <br/>
          给你买玩具、食品、衣服，免费；
          <br/>
          <br/>
          儿子，当你把这些都加到一起时，妈妈付出的所有的爱都是免费的。
          <br/>
          <br/>
          看完之后，儿子的眼睛里噙满了泪水。
          <br/>
          <br/>
          他望着妈妈说：“妈妈，谢谢你做的一切，我真的很爱你……”
          <br/>
          <br/>
          “
          <strong>
           母爱
          </strong>
          ”这笔帐，是任何人都无法算清的，也是无法偿还的！
          <br/>
          <br/>
          <strong>
           六、粗茶淡饭的，是夫妻关系
          </strong>
          <br/>
          <br/>
          不管钱多钱少，只要夫妻和睦，有说有笑，相敬如宾。
          <br/>
          <br/>
          家若温馨，粗茶淡饭也香甜；家若清冷，山珍海味也难咽。
          <br/>
          <br/>
          家庭不在于富有，而在于和睦；生活不在于复杂，而在于简单；幸福不在于明天，而在于眼前。
          <br/>
          <br/>
          夫妻过日子，靠的就是“恩爱”两个字！
