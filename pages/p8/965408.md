 <p>
  现在社会汽车普及，早已成为家家户户必备的代步工具。但有看到各种车祸的新闻报导，总是让人感到触目惊心。看似遥远的各种事故，其实在行车路上多少都会遇到，只是不知哪天意外会落到自己头上而已。
  <span id="hideid" name="hideid" style="color:red;display:none;">
   <span href="https://www.secretchina.com">
   </span>
  </span>
 </p>
 <p>
  这些事故看上去很可怕，但只要预防做到位，那么安全也就大大提升了。对开车而言，
  <span href="https://www.secretchina.com/news/gb/tag/刹车" target="_blank">
   刹车
  </span>
  失灵是很可怕的一件事，若是发生在高速路上，就更加危险了。
 </p>
 <p>
  当然大家也不用太过担心，毕竟刹车
  <span href="https://www.secretchina.com/news/gb/tag/失灵" target="_blank">
   失灵
  </span>
  的几率很低；而且就算刹车真的失灵，也有办法处理。笔者今天就在这里教大家刹车发生状况的保命技巧，你知道几招呢？
 </p>
 <p>
  <strong>
   1
  </strong>
  <strong>
   、镇定
  </strong>
 </p>
 <p>
  这是正确处理任何意外的前提，一旦发觉刹车失灵，千万别惊慌失措，否则误踩油门或乱打方向都只会火上加油。正确办法是握好方向盘，不随意变换车道，并密切注视周围行车情况，保持车辆的平稳运行。很重要的一点，绝对别再踩油门了。
 </p>
 <p>
  <strong>
   2
  </strong>
  <strong>
   、启动应急措施
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  在维持车辆平稳运行时，也应启动汽车应急
  <strong>
   <span href="https://www.secretchina.com/news/gb/tag/双闪" target="_blank">
    双闪
   </span>
  </strong>
  功能，这样能引起周边车主的注意，让大家知道我们的车处于紧急状态。这些车主也会对我们的车况保持警惕状态，不是让我们，就是减速保持距离，不然就是绕道而行。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  总之，周围汽车的礼让，会让我们在刹车失灵的情况下容易驾驶得多，而不至于使我们在应付刹车失灵的情况下，还要分心应付周围可能发生的危险情况。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  <strong>
   3
  </strong>
  <strong>
   、逐步
   <span href="https://www.secretchina.com/news/gb/tag/减档" target="_blank">
    减档
   </span>
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 </center>
 <p>
  在刹车失灵的情况下，要让汽车的速度尽快慢下来的唯一办法，就是按顺序逐步减档。这也是最有效的方法。因为发动机因降档而产生的阻矩力，能迫使发动机转速降低，进而使得汽车速度也快速降低。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">
   </div>
  </div>
  <div>
   <center>
    <div id="div-gpt-ad-1589559869784-0">

   </center>
  </div>
 </center>
 <p>
  正确的办法是在发现刹车失灵时，若是开手排车，就开始依当时的档位循序往下降，不要操之过急。当车子降到1档时，车速应该也降下来了。若是开自排车，那情况就要复杂一些，得先将档位降到3档，然后2档，再到L档。这是利用低档位，强行将车速降低。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  <strong>
   4
  </strong>
  <strong>
   、间歇而短暂运用手刹
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

 <p>
  千万要注意，不是一拉手刹就行了，而是要间歇性的短暂拉手刹。车辆降档，随着车速下降后，就开始间歇性短暂使用手刹，这样能让车辆最终完全停下来。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 </center>
 <p>
  切记，手刹一定要迅速拉起，然后立即放下，动作要快。因为如果强行拉死手刹，会导致手刹线断裂，甚至是车辆失控。所以只能通过频繁拉手刹的动作，强迫汽车自动降档。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

  </ins>
 </center>
 <p>
  <strong>
   5
  </strong>
  <strong>
   、擦撞水马或护栏
  </strong>
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">
   <div id="taboola-midarticle-thumbnails" style="max-width: 632px;height:180px;overflow: hidden;overflow-x: hidden;">

  </div>
  <div>

 </center>
 <p>
  当避无可避时，如果发现路上有水马（又称充水式护栏，或称灌水式活动隔栏，多以鲜色及厚塑料制造，内里注水后能变成不易搬动的护栏）等防撞击设施，要善加利用。因为这些设施里面装了大量的沙子和水，能缓冲撞击力度。
 </p>
 <center>
  <div style="max-width: 632px;height:180px; display: none; text-align: center; margin: 0 auto; overflow: hidden;overflow-x: hidden;">

 <p>
  此外也可利用汽车保险杠、车厢等较坚固的部位，与路边的天然障碍物（例如岩石、大树、高速护栏或土坡等）尽可能的碰撞、摩擦，达到强行停止汽车，尽快脱离险境的目的，也能减少事故损失。
 </p>
